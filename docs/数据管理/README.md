## 📊 功能概览

### 核心能力

```
数据管理系统
├── 股票列表管理（localStorage）
├── 板块基础信息管理（localStorage）
└── 成分股数据管理（IndexedDB）
    ├── 导出数据（JSON）
    ├── 导入数据（JSON）
    └── 清空数据
```

### 技术特性

- ✅ **多存储方案** - localStorage + IndexedDB，根据数据量选择最优方案
- ✅ **状态监控** - 实时显示数据量、更新时间和缓存状态
- ✅ **一键操作** - 快速刷新、清除、备份数据
- ✅ **数据备份** - 支持导出/导入 JSON 格式
- ✅ **自动过期** - 智能判断缓存是否过期
- ✅ **确认保护** - 危险操作需要二次确认

## 💡 使用建议

### 对于普通用户

1. 阅读 [01-使用指南](./01-使用指南.md)
2. 了解各模块的功能和操作
3. 定期备份重要数据（成分股）
4. 遇到问题先尝试清除缓存

### 对于开发者

1. 先读 [02-技术实现](./02-技术实现.md) 的整体架构
2. 理解 localStorage 和 IndexedDB 的使用场景
3. 学习数据刷新和导入导出的实现
4. 需要扩展时参考"扩展指南"章节
