# 股票数据文件格式升级报告

## 📅 变更日期

2026-05-07

## 🎯 变更目标

将股票数据文件从 `.txt` 格式升级为 `.json` 格式,使数据结构更规范、更易解析。

## 📊 变更内容

### 1. 文件格式对比

#### ❌ 旧格式 (.txt)

```
{
    "code": "SH603017",
    "name": "中衡设计",
    "dailyLines": [...],
    "latestQuote": {...},
    "updatedAt": 1777530877865
}

日期买点：2025/11/24，2026/01/07，2026/04/17
```

**问题:**

- JSON 和非 JSON 内容混在一起
- 需要特殊解析逻辑提取日期买点
- 文件扩展名与实际内容不符

#### ✅ 新格式 (.json)

```json
{
    "data": {
        "code": "SH603017",
        "name": "中衡设计",
        "dailyLines": [...],
        "latestQuote": {...},
        "updatedAt": 1777530877865
    },
    "buypointDate": [
        "2025/11/24",
        "2026/01/07",
        "2026/04/17"
    ]
}
```

**优势:**

- 纯 JSON 格式,结构清晰
- 日期买点作为数组字段,易于解析
- 符合标准 JSON 规范
- 便于后续扩展其他元数据
