## 🎯 使用指南

### 如何运行各版本？

```bash
# v1.0 - 原始版本（不推荐，仅用于参考）
node v1.0_ml_buypoint_model_original.cjs

# v2.0 - 参数优化版（适合中等样本量）
node v2.0_ml_buypoint_model_optimized.cjs

# v3.0 - 增强版（当前推荐，最佳性能）
node v3.0_ml_buypoint_model_enhanced.cjs
```

### 如何选择版本？

- **样本量 < 30 个**: 使用 v2.0（深度 7-8 足够）
- **样本量 30-100 个**: 使用 v3.0（深度 12，完美召回）
- **样本量 > 100 个**: 可能需要重新评估，考虑分层模型

### 如何添加新样本？

1. 将 K 线数据保存到 `../stock_data/{股票名称}.txt`
2. 在文件末尾添加：`日期买点：YYYY-MM-DD, YYYY-MM-DD, ...`
3. 编辑 `v3.0_ml_buypoint_model_enhanced.cjs` 的 `stockFiles` 配置
4. 运行脚本重新训练
