## 🚀 使用方法

### 1. 训练模型（推荐使用 v3.0）

```bash
# v3.0 - 增强版（当前推荐，最佳性能）
node v3.0_ml_buypoint_model_enhanced.cjs
```

**功能:**

- 从 28 只股票的 75 个买点样本中提取特征
- 随机选取 270 个非买点作为负样本
- 训练深度 12 的决策树分类模型
- 输出 15 种配置的详细性能对比
- 自动选择最优配置并保存模型

**输出示例:**

```
正样本(买点): 75个
负样本(非买点): 270个
总样本数: 345个

【🏆 推荐配置】
配置ID: Config_14
树深度: 12
召回率: 100.0%
F1分数: 0.99
精确率: 98.7%
准确率: 99.7%
```

### 2. 运行其他版本（可选）

```bash

node v2.0_ml_buypoint_model_optimized.cjs

node v1.0_ml_buypoint_model_original.cjs
```

### 2. 运行分析工具

```bash

# 时间序列分析
```

## 📊 当前模型性能（v3.0）

### 训练数据

- **正样本**: 75 个买点（28 只股票）

  - 原有 6 只: 中衡设计、起帆股份、永杉锂业、山东玻纤、宏昌电子、三孚股份
  - 新增 11 只: 兆新股份、吉林化纤、太极实业、安彩高科、晶科科技、江苏有线、浙富控股、浙文互联、风范股份、盛洋科技、三维通信
  - 全面测试 11 只: 中天科技、先导基电、利通电子、快克智能、环旭电子、赛腾股份、通富微电、金安国纪、风华高科、鼎胜新材、艾华集团

- **负样本**: 270 个随机非买点（每只股票 10 个）

### 模型指标

| 指标    | 数值   | 说明                       |
| ------- | ------ | -------------------------- |
| 准确率  | 99.7%  | 整体预测准确度             |
| 精确率  | 98.7%  | 预测为买点的几乎都是真买点 |
| 召回率  | 100.0% | 捕获所有 75 个买点 ⭐      |
| F1 分数 | 0.99   | 综合评分（接近完美）       |
| 特异性  | 100.0% | 完美识别非买点             |

### 最优配置

- **树深度**: 12
- **负样本数**: 270（每只股票 10 个）
- **配置 ID**: Config_14

### 关键特征

模型学习到的最重要特征：

1. **距高点%** - 距离 60 日高点的百分比
2. **5 日/10 日变化** - 短期价格趋势
3. **布林带宽度** - 波动率指标
4. **ATR%** - 平均真实波幅
5. **MA5/MA20 偏离** - 均线偏离度
6. **量比** - 成交量相对变化

## 🔄 如何增加新样本

### 步骤 1: 收集新买点

当您发现新的优质买点时,记录:

- 股票代码
- 买入日期
- 次日涨幅(验证是否为有效买点)

### 步骤 2: 准备数据文件

将股票 K 线数据保存为 `{股票名称}.txt` 格式,放在 `../stock_data/` 目录下。

### 步骤 3: 更新配置

编辑 `v3.0_ml_buypoint_model_enhanced.cjs` 中的 `stockFiles` 数组:

```javascript
const stockFiles = [
  { file: '中衡设计.txt', dates: ['2026-04-17', '2026-01-07', '2025-11-24'] },
  { file: '起帆股份.txt', dates: ['2026-04-14', '2026-01-16', '2025-12-11'] },
  // ... 添加新股票
  { file: '新股票.txt', dates: ['2026-XX-XX'] },
];
```

### 步骤 4: 重新训练

```bash
node v3.0_ml_buypoint_model_enhanced.cjs
```

模型会自动使用新数据重新训练，生成更新的 `v3.0_buypoint_model_current.json`。

## 💡 后续集成计划

当样本量达到 **100+个买点** 时，将模型集成到项目回测系统中：

### 集成步骤

1. **加载模型**: 在回测 worker 中读取 `buypoint_model.json`
2. **特征提取**: 对每只股票实时计算 9 个特征
3. **预测判断**: 调用模型预测是否为潜在买点
4. **结果展示**: 在回测结果中标记 AI 识别的买点

### 预期效果

- 自动化识别类似历史买点的机会
- 减少人工筛选工作量
- 随着样本增加,准确率持续提升
