## 🚀 快速开始

### 1️⃣ 查看模型性能

```bash
cd ml_model
node ml_buypoint_model.cjs
```

### 2️⃣ 添加新样本

1. 将新股票的 K 线数据放入 `stock_data/`
2. 编辑 `ml_model/ml_buypoint_model.cjs` 中的配置
3. 重新训练模型

### 3️⃣ 验证买点

```bash
cd validation_scripts
node check_xxx_dates.cjs      # 检查日期
node verify_xxx_full.cjs      # 完整验证
```
