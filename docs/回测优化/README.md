## 🚀 快速开始

### 1️⃣ 训练模型

```bash
cd 模型训练
node v3.0_ml_buypoint_model_enhanced.cjs
```

### 2️⃣ 添加新样本

1. 将新股票的 K 线数据放入 `股票数据/`
2. 在文件末尾添加买点日期：`日期买点：YYYY-MM-DD, ...`
3. 编辑 `模型训练/v3.0_ml_buypoint_model_enhanced.cjs` 中的 `stockFiles` 配置
4. 重新训练模型

### 3️⃣ 验证买点覆盖率

```bash
cd 验证脚本
node verify-buypoint-coverage.cjs      # 验证覆盖率
node analyze-missed-buypoints.cjs      # 分析未覆盖买点
```

### 4️⃣ 导出回测数据

1. 在历史回测页面执行全量回测
2. 点击“导出数据”按钮
3. 文件自动保存到 `历史回测数据/` 目录
4. 更新验证脚本中的文件名，然后运行验证
