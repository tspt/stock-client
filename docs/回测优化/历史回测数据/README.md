# 历史回测数据导出目录

## 说明

此目录用于存储从历史回测页面导出的回测数据文件。

## 目录结构

```
历史回测数据/
├── backtest_export_latest.json  # 最新导出文件（每次覆盖更新）
└── README.md                     # 本说明文档
```

**注意**: 使用固定文件名 `backtest_export_latest.json`，每次导出时会自动覆盖旧内容。

## 文件格式

### JSON 格式

- 完整保留数据结构
- 包含元数据（导出时间、股票数量、信号数量）
- 适合程序化处理和数据分析

### CSV 格式

- 表格化展示
- 可直接用 Excel 打开
- 适合快速查看和分享

## 使用场景

1. **模型验证**：验证 ML 模型的回测准确性
2. **数据分析**：进行统计分析和可视化
3. **策略优化**：基于历史数据优化交易策略
4. **报告生成**：生成回测报告和分析文档

## 注意事项

- **固定文件名**: 使用 `backtest_export_latest.json`，每次导出自动覆盖
- **无需管理**: 不需要手动清理旧文件，始终只有一个最新文件
- **验证脚本**: 验证脚本会自动读取最新的导出文件
- **重要数据**: 如需保留历史版本，请手动备份到其他位置

## 相关功能

- 导出功能位于：历史回测页面 → "导出所有数据"按钮
- 股票列表管理：`src/config/exportStocksConfig.ts`
- 动态刷新：在导出模态框中点击"从目录刷新"
