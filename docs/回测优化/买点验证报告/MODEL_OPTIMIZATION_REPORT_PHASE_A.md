## 🔍 决策树规则分析

### 主要分裂特征

根据保存的模型文件，决策树的主要判断逻辑：

**根节点**: `ma20Deviation <= 6.46`

- 如果 MA20 偏离度 <= 6.46%，进入左分支
- 否则进入右分支

**第二层**: `ma5Deviation <= 2.32`

- 如果 MA5 偏离度 <= 2.32%，继续左分支
- 否则进入右分支

**关键特征排序**（按出现频率）:

1. `ma20Deviation` - MA20 偏离度（最重要）
2. `ma5Deviation` - MA5 偏离度
3. `atrPercent` - ATR 波动率
4. `distFromHigh` - 距高点距离
5. `volumeRatio` - 成交量比率

### 规则解读

**买点特征**:

- MA20 偏离度适中（不太高也不太低）
- MA5 偏离度较小（温和上涨）
- ATR 波动率正常
- 成交量不需要特别放大

**这与之前的分析一致**：

- 未覆盖买点是"短期突破型"
- 不需要大幅放量
- 关键是均线的相对位置
