## 💡 核心洞察

### 洞察 1: 未覆盖买点是"短期突破型"

**特征**：

- ✅ 更多站在 5 日均线上方（72.2% vs 51.7%）
- ❌ 较少站在 10 日、20 日均线上方
- ✅ 成交量不弱（1.04x）
- ✅ 涨幅不小（+0.67%）

**典型场景**：

```
价格突然从下方突破5日均线
但10日、20日均线仍在上方压制
属于"超跌反弹"或"短期突破"
而非"趋势确认后的买入"
```

### 洞察 2: 模型偏向"趋势确认后买入"

**已覆盖买点的特征**：

- 更多站在 10 日、20 日均线上方
- RSI 略高（50.57 vs 45.65）
- 说明是趋势已经确立的买点

**模型偏好**：

- ✅ 右侧交易（趋势确认后）
- ❌ 左侧交易（抄底/抢反弹）

### 洞察 3: 涨跌幅和成交量不是关键因素

** surprising finding **：

- 未覆盖买点的涨幅和成交量都不比已覆盖的低
- 说明模型的问题不在这些指标上
- **问题在于均线的组合判断**

## 🎯 优化方向

### 优先级 1: 调整均线组合规则

**当前问题**：

- 可能要求同时站上多条均线
- 或者对长期均线权重过高

**优化建议**：

```javascript
// 当前可能的规则（推测）
if (price > SMA5 && price > SMA10 && price > SMA20) {
  // 触发信号
}

// 建议改为评分制
let score = 0;
if (price > SMA5) score += 1;
if (price > SMA10) score += 1;
if (price > SMA20) score += 1;
if (score >= 2) {
  // 触发信号（站上任意2条均线即可）
}
```

### 优先级 2: 新增"短期突破"规则

**适用场景**：

- 价格突破 5 日均线
- 成交量放大（>1.0x）
- RSI 从低位回升（40-50 区间）
- 即使 10 日、20 日均线仍在上方

**规则示例**：

```javascript
if (price > SMA5 && volumeRatio > 1.0 && RSI > 40 && RSI < 55 && price < SMA10) {
  // 允许在10日均线下方
  // 短期突破信号
}
```

### 优先级 3: 新增"超跌反弹"规则

**适用场景**：

- RSI < 45（相对弱势）
- 价格远离 20 日均线
- 出现阳线反转
- 成交量温和放大

**规则示例**：

```javascript
if (
  RSI < 45 &&
  price < SMA20 * 0.95 && // 远低于20日均线
  isBullish &&
  volumeRatio > 0.9
) {
  // 超跌反弹信号
}
```

### 优先级 4: 实现多指标投票机制

**核心思想**：

- 不再要求所有条件同时满足
- 采用加权评分，达到阈值即触发
- 更灵活，适应性更强

**评分示例**：

```javascript
let score = 0;

// 价格形态（权重3）
if (isBullish) score += 1;
if (changePercent > 1) score += 1;
if (bodySize > 0.2) score += 1;

// 均线关系（权重3）
if (price > SMA5) score += 1;
if (price > SMA10) score += 1;
if (price > SMA20) score += 1;

// 成交量（权重2）
if (volumeRatio > 1.2) score += 2;
else if (volumeRatio > 1.0) score += 1;

// RSI（权重2）
if (RSI > 40 && RSI < 60) score += 1;
if (RSI < 30) score += 2; // 超卖加分

// 触发阈值
if (score >= 6) {
  // 生成买入信号
}
```
