## 📊 功能概览

### 核心能力

```
历史回测系统
├── 数据管理
│   ├── IndexedDB 历史数据存储
│   └── 股票筛选配置
├── 回测引擎
│   ├── Web Worker 并行计算
│   ├── ML 买点模型集成
│   └── 信号识别与验证
├── 结果展示
│   ├── 按股票分组展示
│   ├── 胜率统计
│   └── 收益分析
└── 数据导出
    ├── 单股票导出
    └── 批量导出
```

### 技术特性

- ✅ **本地回测** - 基于 IndexedDB 全量历史数据，无需联网
- ✅ **并行计算** - Web Worker 多线程处理，提升回测速度
- ✅ **ML 模型集成** - 集成机器学习买点模型（v4.0）
- ✅ **智能筛选** - 支持市场类型、名称类型、时间范围筛选
- ✅ **结果持久化** - 回测结果自动保存到 IndexedDB
- ✅ **灵活导出** - 支持单股票和批量导出为 JSON 格式
