# 成分股大全

成分股大全功能提供 A 股市场所有板块（包括行业板块和概念板块）的成分股全景数据，通过浏览器预加载方式避免接口失败问题。

## 📚 文档导航

- [使用指南](./01-使用指南.md) - 面向普通用户的功能介绍和使用方法
- [技术实现](./02-技术实现.md) - 面向开发者的技术架构和实现细节

## 🎯 核心特性

- ✅ **全景获取**：一次性获取所有板块成分股数据
- ✅ **防失败机制**：通过浏览器预加载避免 500 错误
- ✅ **分类展示**：按行业和概念分类展示成分股
- ✅ **数据导出**：支持将成分股数据导出为 CSV/JSON 格式

## 🔧 快速开始

### 1. 浏览器预加载（推荐）

为避免 API 请求失败，建议先执行浏览器预加载：

1. **手动打开浏览器**：在您的默认浏览器中新建标签页
2. **访问预加载地址**：输入或粘贴 `https://quote.eastmoney.com/center/hszs.html`
3. **等待页面加载**：等待页面完全加载完成（约 10-30 秒）
4. **保持页面打开**：建议保持该页面在后台打开，不要关闭
5. **返回应用使用**：切换回股票分析应用继续使用

### 2. 查看成分股数据

1. 选择板块类型（行业板块/概念板块）
2. 使用搜索功能查找特定股票或板块
3. 浏览成分股列表和详细数据
4. 按需导出数据进行分析

## 📊 数据说明

### 板块分类

- **行业板块**：基于产业链划分，分类相对稳定
- **概念板块**：基于主题热点划分，变化较为频繁

### 数据字段

- 股票代码和名称
- 所属板块信息
- 实时行情数据（价格、涨跌幅等）
- 资金流向数据（主力净流入等）

## ⚠️ 注意事项

1. **预加载重要性**：首次使用或数据异常时，务必执行浏览器预加载
2. **网络要求**：需要稳定的网络连接才能获取实时数据
3. **数据延迟**：行情数据可能存在秒级延迟，属正常现象
4. **缓存策略**：系统会自动缓存数据，减少重复请求

## 🆘 常见问题

### Q: 为什么需要浏览器预加载？

A: 预加载可以建立浏览器会话和缓存，避免直接 API 请求导致的 500 错误和频率限制。

### Q: 预加载后仍然获取失败怎么办？

A:

1. 确认预加载页面已完全加载
2. 检查网络连接状态
3. 尝试清除浏览器缓存后重新预加载
4. 联系技术支持

### Q: 如何导出成分股数据？

A: 在成分股大全页面点击"导出"按钮，选择 CSV 或 JSON 格式即可下载。
