## 📊 功能概览

### 核心能力

```
分析记录系统
├── 数据统计
│   ├── 股票上榜次数统计
│   ├── 最新上榜日期
│   └── 所属概念/行业
├── 趋势分析
│   ├── 每日上榜数量趋势
│   └── 时间范围筛选
└── 数据管理
    ├── 按日期范围筛选
    └── 清空所有记录
```

### 技术特性

- ✅ **实时统计** - 自动计算股票上榜次数和频率
- ✅ **趋势可视化** - 图表展示每日上榜数量变化
- ✅ **灵活筛选** - 支持按日期范围筛选数据
- ✅ **双视图切换** - 表格视图和趋势图视图
- ✅ **一键清空** - 快速清理所有分析记录
- ✅ **概念标签** - 显示股票所属概念和行业
