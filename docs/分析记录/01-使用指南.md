# 分析记录 - 使用指南

> 📚 **相关文档**: [技术实现](./02-技术实现.md) | [返回上级](./README.md)

本文档面向**普通用户**，介绍分析记录功能的使用方法和数据解读。

---

## 🎯 功能概览

分析记录页面展示股票在机会分析中的上榜统计信息和趋势图，帮助用户追踪热门股票、分析市场热点和发现潜在投资机会。

### 核心能力

- ✅ **上榜统计** - 自动统计每只股票的上榜次数和频率
- ✅ **趋势可视化** - 图表展示每日上榜数量变化趋势
- ✅ **概念标签** - 显示股票所属的概念板块和行业分类
- ✅ **日期筛选** - 支持按时间范围筛选统计数据
- ✅ **双视图切换** - 表格视图和趋势图视图自由切换
- ✅ **一键清空** - 快速清理所有历史分析记录

## 📋 页面布局

### 1️⃣ 顶部工具栏

**功能组件**：

- 📅 **日期选择器**：选择统计的时间范围
- 🔄 **刷新按钮**：重新加载统计数据
- 🗑️ **清空按钮**：删除所有分析记录

**特性**：自适应不同屏幕尺寸

### 2️⃣ 视图切换标签

**两种视图**：

- 📊 **表格视图**：详细展示每只股票的统计信息
- 📈 **趋势图视图**：图表展示每日上榜数量变化

**交互**：点击标签切换视图

### 3️⃣ 数据展示区域

#### 表格视图

**展示字段**：

| 字段         | 含义             | 说明           |
| ------------ | ---------------- | -------------- |
| 股票代码     | 股票唯一标识     | 6 位数字代码   |
| 股票名称     | 股票中文名称     | 便于识别       |
| 出现次数     | 上榜总次数       | 反映股票热度   |
| 最新上榜日期 | 最后一次上榜日期 | 反映近期活跃度 |
| 所属概念     | 概念板块标签     | 最多显示 3 个  |
| 所属行业     | 行业分类         | 证监会行业分类 |

**排序功能**：

- 点击列标题可按该列排序
- 支持升序/降序切换

#### 趋势图视图

**展示内容**：

- 📅 **横轴**：日期（最近 30 天）
- 📊 **纵轴**：当日上榜股票数量
- 📈 **折线**：连接各日数据点

**交互**：

- 悬停查看具体数值
- 缩放查看细节

## ⚙️ 使用方法

### 查看统计数据

1. **进入分析记录页面**
2. **默认显示表格视图**
3. **浏览股票列表**
4. **点击列标题排序**

**解读要点**：

- **出现次数多**：表示该股票频繁出现在机会分析中，可能是热门股
- **最新上榜日期近**：表示该股票近期活跃
- **概念标签**：帮助理解股票所属的热点主题

### 筛选时间范围

1. **点击顶部的日期选择器**
2. **选择起始日期和结束日期**
3. **系统自动重新计算统计数据**

**使用场景**：

- 查看特定时间段的市场热点
- 对比不同时期的股票表现
- 分析季节性规律

**建议**：

- 短期分析：选择最近 7-14 天
- 中期分析：选择最近 30-60 天
- 长期分析：选择最近 90-180 天

### 切换视图

1. **点击顶部的标签页**
2. **选择"表格"或"趋势图"**
3. **查看对应的数据展示**

**表格视图适用场景**：

- 需要查看详细的股票信息
- 需要排序和筛选
- 需要导出或复制数据

**趋势图视图适用场景**：

- 需要了解整体趋势
- 需要发现周期性规律
- 需要直观展示数据变化

### 清空分析记录

1. **点击"清空"按钮**
2. **确认操作**（弹出确认对话框）
3. **等待清空完成**

**注意事项**：

- ⚠️ **此操作不可恢复**
- 会删除所有历史分析记录
- 建议在清空前导出重要数据

**使用场景**：

- 开始新的分析周期
- 清理过期数据
- 释放存储空间

## 📊 数据解读

### 出现次数

**含义**：股票在机会分析中出现的总次数

**解读**：

- **高频股票**（>10 次）：市场热点，关注度高
- **中频股票**（3-10 次）：有一定关注度
- **低频股票**（1-2 次）：偶尔出现，可能是新机会

**应用**：

- 高频股票：跟踪其走势，寻找买卖点
- 中频股票：观察是否有持续热度
- 低频股票：可能是潜在的新机会

### 最新上榜日期

**含义**：股票最后一次出现在机会分析中的日期

**解读**：

- **近期**（3 天内）：当前活跃，值得重点关注
- **中期**（3-7 天）：热度有所下降，但仍有机会
- **远期**（7 天以上）：可能需要重新评估

**应用**：

- 结合出现次数判断股票热度
- 优先关注近期活跃的股票
- 跟踪热度变化趋势

### 所属概念

**含义**：股票所属的概念板块（如：人工智能、新能源汽车等）

**解读**：

- **热门概念**：多个股票同时出现，表示该概念是市场热点
- **冷门概念**：少数股票出现，可能是细分领域机会

**应用**：

- 识别市场主线热点
- 发现新兴概念板块
- 进行板块轮动分析

### 趋势图

**含义**：每日上榜股票数量的变化趋势

**解读**：

- **上升趋势**：市场热度增加，机会增多
- **下降趋势**：市场热度降低，机会减少
- **波动趋势**：市场震荡，需要精选个股

**应用**：

- 判断市场整体情绪
- 选择合适的入场时机
- 调整仓位配置策略

## ❓ 常见问题

### Q1: 为什么有些股票没有出现次数？

**A**: 可能的原因：

1. **该股票从未出现在机会分析中**
2. **时间范围筛选排除了相关记录**
3. **数据已被清空**

**解决方案**：

- 扩大时间范围
- 在机会分析页面触发扫描
- 检查是否有筛选条件限制

### Q2: 趋势图为什么不显示数据？

**A**: 可能的原因：

1. **所选时间范围内没有数据**
2. **所有记录已被清空**
3. **浏览器兼容性问题**

**解决方案**：

- 调整时间范围
- 确认有分析记录
- 刷新页面重试

### Q3: 如何导出分析记录？

**A**: 目前不支持直接导出功能。如果需要保存数据：

1. **截图保存**：截取表格或趋势图
2. **手动记录**：记录重要的股票信息
3. **等待更新**：后续版本可能添加导出功能

### Q4: 分析记录会保存多久？

**A**:

- 记录永久保存在 IndexedDB 中
- 除非手动清空或清除浏览器数据
- 建议定期清理过期记录

### Q5: 如何提高统计准确性？

**A**: 建议：

1. **定期执行机会分析**：确保数据完整
2. **合理设置时间范围**：避免数据过多或过少
3. **及时清理过期数据**：保持数据新鲜度
4. **交叉验证**：结合其他指标综合判断

### Q6: 概念标签不准确怎么办？

**A**:

- 概念数据来自机会分析时的快照
- 如果概念发生变化，需要重新分析
- 可以在机会分析页面重新触发扫描
