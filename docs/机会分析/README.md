## 📊 功能概览

### 核心筛选能力

```
机会分析筛选体系
├── 基础数据筛选
│   ├── 价格、市值、换手率、市盈率
│   ├── KDJ-J、RSI（可配置周期）
│   └── 涨跌停统计
│
├── 横盘筛选
│   ├── 检索根数、连续根数
│   ├── 波动阈值、MA10 约束
│   └── 强度评分
│
├── 趋势线筛选
│   ├── 检索根数 M、连续根数 N
│   └── MA5 站立判定
│
├── 单日异动筛选
│   ├── 急跌/急涨
│   ├── V型/A型反转
│   └── W底/M顶形态
│
└── 形态筛选
    ├── 技术指标：MACD、布林带
    ├── K 线形态：13 种经典形态
    └── 趋势形态：5 种趋势识别
```

### 技术特性

- ✅ **纯前端筛选**：首次分析后无需联网
- ✅ **Web Worker**：后台计算，不卡顿 UI
- ✅ **智能重算**：只计算激活的指标
- ✅ **偏好记忆**：自动保存筛选条件
- ✅ **数据持久化**：IndexedDB + localStorage

## 💡 使用建议

### 对于普通用户

1. 阅读 [01-使用指南](./01-使用指南.md)
2. 了解各筛选条件的含义和用法
3. 参考"推荐筛选组合"快速上手
4. 遇到问题查看"常见问题"章节

### 对于开发者

1. 先读 [02-技术实现](./02-技术实现.md) 的整体架构
2. 理解数据流和双存储策略
3. 学习筛选引擎的两阶段过滤
4. 需要扩展时参考"扩展新筛选项"章节
