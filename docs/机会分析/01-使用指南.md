# 机会分析 - 使用指南

> 📚 **相关文档**: [技术实现](./02-技术实现.md) | [返回上级](./README.md)

本文档面向**普通用户**，介绍机会分析功能的使用方法和参数配置。

---

## 🎯 功能概览

机会分析是一个强大的股票筛选工具，支持多维度条件组合筛选，帮助您快速发现潜在投资机会。

### 核心能力

- ✅ **基础数据筛选**：价格、市值、换手率、市盈率等
- ✅ **技术指标筛选**：RSI、MACD、布林带
- ✅ **形态识别**：13 种 K 线形态 + 5 种趋势形态
- ✅ **横盘检测**：智能识别横盘整理阶段
- ✅ **趋势线分析**：识别上升趋势
- ✅ **单日异动**：捕捉急涨急跌机会
- ✅ **AI 智能分析**：趋势预测、多维评分、相似形态匹配
- ✅ **纯前端筛选**：无需联网，毫秒级响应
- ✅ **偏好记忆**：自动保存筛选条件

## 📋 筛选条件详解

### 1️⃣ 数据筛选面板

#### 基础指标

| 筛选项   | 说明            | 默认值    | 建议               |
| -------- | --------------- | --------- | ------------------ |
| 价格范围 | 股票现价范围    | 3~30 元   | 根据风险偏好调整   |
| 市值范围 | 流通市值（亿）  | 30~500 亿 | 中小盘股波动更大   |
| 换手率   | 当日换手率（%） | ≥1%       | 活跃股通常 >3%     |
| 市盈率   | PE 比值         | 不限      | 价值投资可设上限   |
| KDJ-J    | KDJ 指标的 J 值 | 不限      | <0 超卖，>100 超买 |

#### RSI（相对强弱指数）

**什么是 RSI？**

- 衡量价格变动速度的动量指标
- 范围：0-100
- **RSI < 30**：超卖区域，可能反弹
- **RSI > 70**：超买区域，可能回调

**可配置周期**：

- **6 日**（默认）：短周期，敏感度高，适合短线
- **12 日**：中周期，平衡型
- **24 日**：长周期，稳定型，适合中长线

**使用方法**：

1. 设置 RSI 范围（如：30~70）
2. 选择 RSI 周期（6/12/24 日）
3. 系统会计算最新交易日的 RSI 值并筛选

**数据要求**：至少需要 `周期 + 1` 根 K 线

---

#### 涨跌停统计

| 筛选项       | 说明               | 示例  |
| ------------ | ------------------ | ----- |
| 最近涨停数   | 指定周期内涨停次数 | ≥2 次 |
| 涨停周期范围 | 统计的天数范围     | 20 天 |
| 最近跌停数   | 指定周期内跌停次数 | ≥1 次 |
| 跌停周期范围 | 统计的天数范围     | 20 天 |

**用途**：捕捉强势股或超跌股

### 2️⃣ 横盘筛选面板

**什么是横盘？**
股价在一定范围内波动，没有明显的上涨或下跌趋势，通常是变盘前兆。

#### 参数说明

| 参数        | 含义                     | 默认值 | 建议                    |
| ----------- | ------------------------ | ------ | ----------------------- |
| 检索根数    | 从最新 K 线向前看多少根  | 10     | 日线：10-20，周线：5-10 |
| 连续根数    | 需要连续满足横盘的根数   | 5      | 通常为检索根数的一半    |
| 波动阈值(%) | 价格波动幅度上限         | 2%     | 严格：1-2%，宽松：3-5%  |
| MA10 约束   | 收盘价是否需在 MA10 之上 | 可选   | 勾选更严格              |

**判断方法**：

1. **价格波动率法**：(最高价 - 最低价) / 平均价 < 阈值
2. **MA 收敛法**：多条均线靠拢程度

**强度评分**：0-100 分，分数越高横盘越明显

**使用建议**：

- **严格模式**：波动阈值 1-2%，只看强横盘（强度 >70）
- **中等模式**：波动阈值 2-3%，平衡筛选
- **宽松模式**：波动阈值 3-5%，不遗漏任何横盘

---

### 3️⃣ 趋势线筛选面板

**什么是趋势线？**
识别最近 M 根 K 线中是否存在连续 N 根"沿趋势上行"的走势。

#### 判定规则

每一根 K 线需同时满足：

1. ✅ 收盘价 ≥ 前一日收盘价（不跌）
2. ✅ 收盘价 ≥ 当日 MA5 均线

#### 参数说明

| 参数       | 含义                   | 默认值 | 说明     |
| ---------- | ---------------------- | ------ | -------- |
| 检索根数 M | 从最新向前看的窗口大小 | 10     | 最大 500 |
| 连续根数 N | 需要连续满足的根数     | 5      | 必须 ≤ M |

**结果说明**：

- **含尾**：命中段延伸到最新一根 K 线（最理想）
- **非尾**：命中段在窗口内但未到最新

**使用场景**：

- 寻找刚刚启动的上升趋势
- 配合横盘筛选：先横盘后突破

---

### 4️⃣ 单日异动筛选面板

**什么是单日异动？**
捕捉短期内的大幅波动模式，如急涨、急跌、V 型反转等。

#### 参数说明

| 参数      | 含义               | 默认值 | 说明       |
| --------- | ------------------ | ------ | ---------- |
| 检索根数  | 从最新向前看的窗口 | 20 根  | 最大 500   |
| 阈值 M(%) | 单日涨跌幅阈值     | 6%     | 越大越严格 |

#### 异动模式（可多选，OR 关系）

| 模式         | 说明          | 适用场景     |
| ------------ | ------------- | ------------ |
| 仅急跌       | 单日跌幅 ≥ M% | 抄底机会     |
| 仅急涨       | 单日涨幅 ≥ M% | 追涨机会     |
| 急跌 → 急涨  | V 型反转      | 底部反转信号 |
| 急涨 → 急跌  | A 型回落      | 顶部警示     |
| 急跌横盘急涨 | W 底形态      | 强力底部信号 |
| 急涨横盘急跌 | M 顶形态      | 强力顶部信号 |

**使用技巧**：

- 急跌后横盘再急涨：可能是主力洗盘后的拉升
- 急涨后横盘再急跌：可能是主力出货

### 5️⃣ 形态筛选面板

#### 技术指标

##### MACD（平滑异同移动平均线）

**三种信号**（可多选，OR 关系）：

- ✅ **金叉**：DIF 从下向上穿越 DEA → 看涨信号
- ✅ **死叉**：DIF 从上向下穿越 DEA → 看跌信号
- ✅ **背离**：价格创新高/新低但 DIF 未同步 → 反转信号

**数据要求**：至少 35 根 K 线

##### 布林带（Bollinger Bands）

**三种位置**（可多选，OR 关系）：

- ✅ **上轨附近**：价格接近上轨（5% 以内）→ 可能回调
- ✅ **中轨附近**：价格在中轨附近 → 震荡行情
- ✅ **下轨附近**：价格接近下轨（5% 以内）→ 可能反弹

**数据要求**：至少 20 根 K 线

---

#### K 线形态（13 种）

代码实现了 13 种经典 K 线形态，基于回溯窗口内的任意位置检测。

**单根形态（7 种）**：

| 形态       | 特征                   | 信号        |
| ---------- | ---------------------- | ----------- |
| 锤头线     | 下影线长，实体小       | 底部反转 ↑  |
| 射击之星   | 上影线长，实体小       | 顶部反转 ↓  |
| 十字星     | 开盘 ≈ 收盘            | 犹豫不决 ↔  |
| 倒锤头线   | 上影线长，实体在下部   | 底部反转 ↑  |
| 上吊线     | 下影线长，实体在上部   | 顶部反转 ↓  |
| 蜻蜓十字星 | 开盘=收盘=最高，有下影 | 强烈看涨 ↑↑ |
| 墓碑十字星 | 开盘=收盘=最低，有上影 | 强烈看跌 ↓↓ |

**双根形态（4 种）**：

| 形态     | 特征                      | 信号   |
| -------- | ------------------------- | ------ |
| 阳包阴   | 第二根阳线吞没第一根阴线  | 看涨 ↑ |
| 阴包阳   | 第二根阴线吞没第一根阳线  | 看跌 ↓ |
| 阳孕阴   | 阴线在阳线内部            | 看跌 ↓ |
| 阴孕阳   | 阳线在阴线内部            | 看涨 ↑ |
| 乌云盖顶 | 阴线深入阳线实体 50% 以下 | 看跌 ↓ |
| 刺透形态 | 阳线深入阴线实体 50% 以上 | 看涨 ↑ |

**三根形态（4 种）**：

| 形态     | 特征                 | 信号         |
| -------- | -------------------- | ------------ |
| 早晨之星 | 阴线 → 小实体 → 阳线 | 底部反转 ↑↑  |
| 黄昏之星 | 阳线 → 小实体 → 阴线 | 顶部反转 ↓↓  |
| 三只乌鸦 | 连续三根阴线         | 强烈看跌 ↓↓↓ |
| 三兵红烛 | 连续三根阳线         | 强烈看涨 ↑↑↑ |

**K 线形态检索根数**：默认 20 根，可配置 5-200 根

> 💡 **提示**：虽然代码实现了 13 种形态，但 UI 中目前只暴露了部分形态的筛选选项。

#### 趋势形态（5 种）

| 形态     | 判断方法              | 信号    |
| -------- | --------------------- | ------- |
| 上升趋势 | 线性回归斜率为正      | 看涨 ↑  |
| 下降趋势 | 线性回归斜率为负      | 看跌 ↓  |

### 6️⃣ AI 分析筛选面板

**什么是 AI 分析？**
基于机器学习算法对股票进行多维度智能分析，提供趋势预测、综合评分和相似形态匹配。

> ⚠️ **注意**：需要先点击“一键分析”生成 AI 分析数据后，才能使用此筛选功能。

#### 启用开关

- **启用 AI 分析筛选**：总开关，关闭时不进行 AI 筛选
- 开启后，所有子筛选项才会生效

---

#### 趋势预测

基于历史数据和模式识别，预测未来走势方向。

**预测方向**（可多选，OR 关系）：

| 方向 | 说明     | 适用场景     |
| ---- | -------- | ------------ |
| 看涨 | 预测上涨 | 寻找买入机会 |
| 看跌 | 预测下跌 | 规避风险     |
| 横盘 | 预测震荡 | 观望等待     |

**置信度范围**（0-100%）：

- 表示预测的可信程度
- 建议设置最低置信度（如 ≥70%）提高准确性
- 范围：0~100%，默认不限

**示例配置**：

```
趋势预测：看涨
置信度：70~100%
```

---

#### 智能推荐评分

综合多个维度的评分系统，总分 0-100 分，分数越高越值得关注。

##### 综合评分

- **范围**：0-100 分
- **含义**：整体推荐程度
- **建议**：≥70 分为优质标的

##### 技术面评分

- **评估维度**：技术指标表现（RSI、MACD、均线等）
- **范围**：0-100 分
- **高分含义**：技术面强势

##### 形态评分

- **评估维度**：K 线形态质量
- **范围**：0-100 分
- **高分含义**：出现经典看涨/看跌形态

##### 趋势评分

- **评估维度**：趋势强度和持续性
- **范围**：0-100 分
- **高分含义**：趋势明确且强劲

##### 风险评分

- **评估维度**：波动性、回撤风险等
- **范围**：0-100 分
- **⚠️ 重要**：**分数越高风险越低**（与常规理解相反）
- **建议**：≥70 分表示风险可控

**评分筛选技巧**：

```markdown
# 保守型策略

综合评分：≥80
技术面：≥75
风险评分：≥80

# 激进型策略

综合评分：≥60
技术面：≥60
风险评分：≥50

# 均衡型策略

综合评分：70-90
趋势评分：≥70
风险评分：≥70
```

#### 相似形态匹配

基于历史数据寻找相似的价格形态，并回溯其后续表现。

**筛选选项**：

| 筛选项         | 说明                     | 用途           |
| -------------- | ------------------------ | -------------- |
| 要求有相似形态 | 必须存在历史相似形态     | 确保有参考案例 |
| 最低相似度     | 设置相似度阈值（0-100%） | 控制匹配严格度 |

**相似度说明**：

- **计算方式**：基于价格序列的欧氏距离或相关系数
- **范围**：0-100%，越高表示形态越相似
- **建议阈值**：
  - 严格模式：≥80%（非常相似）
  - 中等模式：≥60%（较为相似）
  - 宽松模式：≥40%（有一定相似性）

**使用场景**：

- 寻找历史上成功突破的形态
- 避免重复失败的形态模式
- 结合历史表现判断成功率

**示例配置**：

```
要求有相似形态：✅ 勾选
最低相似度：70%
```

#### AI 筛选注意事项

1. **数据依赖**：

   - 必须先执行“一键分析”生成 AI 数据
   - 没有 AI 分析数据的股票会被自动过滤

2. **性能影响**：

   - AI 筛选在 Web Worker 中运行，不会卡顿界面
   - 首次分析可能需要几秒到几十秒（取决于股票数量）

3. **筛选逻辑**：

   - 趋势预测：勾选多项为 OR 关系
   - 评分范围：同时满足最小值和最大值（AND 关系）
   - 不同类别间：AND 关系（必须同时满足）

4. **置信度理解**：

   - 置信度 ≠ 准确率
   - 高置信度表示模型对该预测更有把握
   - 建议结合其他指标综合判断

5. **评分解读**：
   - 各项评分独立计算，可能不一致
   - 例如：技术面高分但风险评分低 = 强势但有风险
   - 建议关注综合评分和风险评分的平衡

## 🔧 使用技巧

### 筛选逻辑说明

- **同类别内**：勾选多项为 **OR 关系**（满足任一即可）
  - 例如：同时勾选"金叉"和"背离"，只要满足一个就通过
- **不同类别间**：为 **AND 关系**（必须同时满足）
  - 例如：同时启用"RSI 范围"和"MACD 金叉"，两个都要满足

## ⚠️ 注意事项

### 数据要求

某些指标需要足够的历史数据：

| 指标        | 最少 K 线数 |
| ----------- | ----------- |
| RSI (6 日)  | 7 根        |
| RSI (12 日) | 13 根       |
| RSI (24 日) | 25 根       |
| MACD        | 35 根       |
| 布林带      | 20 根       |
| 双根形态    | 2 根        |
| 三根形态    | 3 根        |
| 趋势形态    | 20 根       |

**建议**：一键分析时默认拉取 300 根 K 线，足够所有指标计算。

### 性能优化

- ✅ **纯前端筛选**：首次分析后，后续筛选无需联网
- ✅ **Web Worker**：复杂计算在后台线程，不卡顿界面
- ✅ **智能重算**：只计算激活的指标，节省资源
- ✅ **防抖处理**：筛选条件变化 300ms 后才重算

### 何时需要重新分析

需要点击"一键分析"的情况：

- ❌ 修改了 K 线周期（日/周/月/年）
- ❌ 修改了 K 线根数
- ❌ 修改了市场范围（沪深主板/创业板）
- ❌ 修改了名称类型（ST/非 ST）

不需要重新分析的情况：

- ✅ 只修改筛选条件（自动重算）
- ✅ 只修改横盘/趋势线参数（自动重算）

## 🆘 常见问题

### Q1: 为什么我的股票没有被筛选出来？

**可能原因**：

1. 某个条件设置过严
2. 数据不足（K 线根数不够）
3. 多个 AND 条件难以同时满足

**解决方法**：

- 逐步放宽条件，观察哪些股票出现
- 检查 K 线数据是否充足（建议 300 根）
- 减少同时启用的筛选类别

### Q2: 如何找到最佳筛选参数？

**建议流程**：

1. 先用宽松条件筛选出候选池
2. 逐步收紧条件，观察结果变化
3. 记录效果好的参数组合
4. 定期回测验证参数有效性

### Q3: 筛选结果可以导出吗？

**支持导出**：

- ✅ Excel 格式：包含所有字段
- ✅ PNG 图片：股票名称列表
- ✅ 点击"导出"按钮即可

### Q4: 为什么刷新后数据还在？

**原因**：

- 分析结果存储在 IndexedDB（浏览器数据库）
- 筛选条件存储在 localStorage
- 两者配合实现纯前端体验
