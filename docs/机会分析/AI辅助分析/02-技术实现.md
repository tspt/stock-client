## 1. 趋势预测实现

### 1.1 算法原理

采用**多指标加权投票机制**，每个技术指标独立给出方向判断和权重，最后综合评分。

```typescript
interface Signal {
  direction: 'up' | 'down' | 'sideways';
  weight: number; // 该信号的权重
  reason: string; // 判断依据
}
```

### 1.2 各指标分析逻辑

#### 移动平均线分析 (权重: 0.3 + 0.25)

```typescript
// 多头排列检测
if (price > MA5 && MA5 > MA10 && MA10 > MA20) {
  signals.push({ direction: 'up', weight: 0.3, reason: '多头排列' });
}

// 金叉检测
if (prevMA5 <= prevMA10 && currMA5 > currMA10) {
  signals.push({ direction: 'up', weight: 0.25, reason: 'MA5上穿MA10' });
}
```

**判断标准：**

- 价格与三条均线的相对位置
- MA5 与 MA10 的交叉情况
- 均线斜率变化

#### RSI 分析 (权重: 0.2 + 0.15)

```typescript
// 超卖区域
if (RSI6 < 30) {
  signals.push({ direction: 'up', weight: 0.2, reason: 'RSI进入超卖区' });
}

// RSI金叉
if (prevRSI6 <= prevRSI12 && currRSI6 > currRSI12) {
  signals.push({ direction: 'up', weight: 0.15, reason: 'RSI金叉' });
}
```

**阈值设置：**

- 超卖区：RSI < 30
- 超买区：RSI > 70
- 使用 RSI(6)和 RSI(12)双周期

#### MACD 分析 (权重: 0.25 + 0.15)

```typescript
// MACD金叉
if (prevDIF <= prevDEA && currDIF > currDEA) {
  signals.push({ direction: 'up', weight: 0.25, reason: 'MACD金叉' });
}

// 红柱放大
if (currMACD > 0 && currMACD > prevMACD) {
  signals.push({ direction: 'up', weight: 0.15, reason: 'MACD红柱放大' });
}
```

**判断维度：**

- DIF 与 DEA 的交叉
- MACD 柱状图的变化趋势

#### 布林带分析 (权重: 0.2)

```typescript
const position = (close - lower) / (upper - lower);

if (position < 0.2) {
  signals.push({ direction: 'up', weight: 0.2, reason: '价格接近下轨' });
} else if (position > 0.8) {
  signals.push({ direction: 'down', weight: 0.2, reason: '价格接近上轨' });
}
```

**位置判断：**

- 下轨附近 (< 20%)：看涨
- 上轨附近 (> 80%)：看跌
- 中轨附近 (40%-60%)：横盘

#### 趋势形态分析 (权重: 0.2-0.3)

复用现有的`trendPatterns.ts`模块：

- 上升趋势：+0.25
- 下降趋势：-0.25
- 突破形态：+0.3
- 跌破形态：-0.3

### 1.3 综合评分算法

```typescript
let upScore = 0,
  downScore = 0,
  sidewaysScore = 0;

signals.forEach((signal) => {
  switch (signal.direction) {
    case 'up':
      upScore += signal.weight;
      break;
    case 'down':
      downScore += signal.weight;
      break;
    case 'sideways':
      sidewaysScore += signal.weight;
      break;
  }
});

const totalScore = upScore + downScore + sidewaysScore;
const confidence = Math.max(upScore, downScore, sidewaysScore) / totalScore;
```

**方向判定：**

- 得分最高的方向为预测方向
- 置信度 = 最高分 / 总分

### 1.4 支撑阻力位计算

```typescript
const supportLevel = Math.min(...recentLows); // 最近20天最低价
const resistanceLevel = Math.max(...recentHighs); // 最近20天最高价
```

### 1.5 目标价预测

```typescript
if (direction === 'up' && resistanceLevel) {
  targetPrice = resistanceLevel; // 上涨目标为阻力位
} else if (direction === 'down' && supportLevel) {
  targetPrice = supportLevel; // 下跌目标为支撑位
}
```

## 2. 相似形态识别实现

### 2.1 特征提取

从 K 线数据中提取 4 维特征向量：

```typescript
interface PatternFeatures {
  priceChanges: number[]; // 价格变化率序列 (19维)
  volumeChanges: number[]; // 成交量变化率序列 (19维)
  volatility: number; // 波动率 (1维)
  trend: number; // 趋势斜率 (1维)
}
```

**价格变化率：**

```typescript
for (let i = 1; i < recent.length; i++) {
  const change = ((close[i] - close[i - 1]) / close[i - 1]) * 100;
  priceChanges.push(change);
}
```

**波动率计算：**

```typescript
const mean = closes.reduce((a, b) => a + b) / n;
const variance = closes.reduce((acc, val) => acc + (val - mean)², 0) / n;
const volatility = sqrt(variance) / mean;
```

**趋势斜率（线性回归）：**

```typescript
slope = (n * Σxy - Σx * Σy) / (n * Σx² - (Σx)²)
```

### 2.2 相似度计算

采用**多维度加权余弦相似度**：

```typescript
similarity =
  priceSimilarity * 0.4 + // 价格变化权重40%
  volumeSimilarity * 0.2 + // 成交量权重20%
  volatilitySimilarity * 0.2 + // 波动率权重20%
  trendSimilarity * 0.2; // 趋势权重20%
```

**余弦相似度：**

```typescript
cosineSimilarity = (A · B) / (||A|| * ||B||)
                 = Σ(ai * bi) / (sqrt(Σai²) * sqrt(Σbi²))
```

**波动率相似度：**

```typescript
volatilitySimilarity = max(0, 1 - |vol1 - vol2| * 10)
```

**趋势相似度：**

```typescript
trendSimilarity = max(0, 1 - |trend1 - trend2| / avgTrend)
```

### 2.3 历史表现计算

```typescript
const startPrice = klineData[len - observationPeriod - 1].close;
const endPrice = klineData[len - 1].close;
const changePercent = ((endPrice - startPrice) / startPrice) * 100;
```

### 2.4 搜索优化

- **搜索范围限制**：默认 100 只股票
- **相似度阈值**：默认 0.7（70%）
- **结果数量**：返回 Top 5
- **自排除**：跳过当前股票本身

## 3. 智能选股推荐实现

### 3.1 评分体系

```
总分 = 技术面 * 0.3 + 形态 * 0.25 + 趋势 * 0.25 + 风险 * 0.2
```

### 3.2 技术面评分 (0-100)

**基础分：50 分**

**加分项：**

- KDJ-J < 20：+15 分（超卖）
- 均线多头排列：+15 分
- 成交量放大 1.5 倍：+10 分

**减分项：**

- KDJ-J > 80：-10 分（超买）
- 均线空头排列：-15 分

```typescript
let score = 50;

if (kdjJ < 20) score += 15;
else if (kdjJ > 80) score -= 10;

if (MA5 > MA10 && MA10 > MA20) score += 15;
else if (MA5 < MA10 && MA10 < MA20) score -= 15;

if (recentVolume > prevVolume * 1.5) score += 10;

return clamp(score, 0, 100);
```

### 3.3 形态评分 (0-100)

**基础分：50 分**

**加分项：**

- 急跌-横盘-急涨形态：+20 分
- 处于横盘整理：+15 分

**减分项：**

- 近期有急跌：-10 分

```typescript
let score = 50;

if (sharpMovePatterns?.dropThenFlatThenRise) score += 20;
if (sharpMovePatterns?.onlyDrop) score -= 10;
if (consolidation?.isConsolidation) score += 15;

return clamp(score, 0, 100);
```

### 3.4 趋势评分 (0-100)

**基础分：50 分**

**加分项：**

- 沿趋势线运行：+20 分
- 今日涨幅 3%-7%：+10 分（动能良好）

**减分项：**

- 今日涨幅>9%：-5 分（警惕回调）
- 今日跌幅<-5%：-10 分

```typescript
let score = 50;

if (trendLine?.isHit) score += 20;

if (changePercent > 3 && changePercent < 7) score += 10;
else if (changePercent > 9) score -= 5;
else if (changePercent < -5) score -= 10;

return clamp(score, 0, 100);
```

### 3.5 风险评分 (0-100，越高风险越低)

**基础分：50 分**

**减分项（增加风险）：**

- ST 股票：-20 分
- 市盈率>100：-15 分
- 换手率>20%：-10 分
- 波动率>5%：-10 分

**加分项（降低风险）：**

- 市盈率<50：+10 分
- 换手率 5%-20%：+5 分

```typescript
let score = 50;

if (name.includes('ST')) score -= 20;

if (peRatio > 100) score -= 15;
else if (peRatio > 0 && peRatio < 50) score += 10;

if (turnoverRate > 20) score -= 10;
else if (turnoverRate > 5 && turnoverRate <= 20) score += 5;

if (volatility > 0.05) score -= 10;

return clamp(score, 0, 100);
```

### 3.6 推荐理由生成

根据各项指标状态自动生成自然语言描述：

```typescript
const reasons: string[] = [];

if (kdjJ < 20) reasons.push('KDJ-J值处于低位，可能超卖');
if (MA5 > MA10 && MA10 > MA20) reasons.push('均线多头排列');
if (consolidation?.isConsolidation) {
  reasons.push(`处于横盘整理状态（${consolidation.matchedTypeLabels.join(', ')}）`);
}

return reasons.slice(0, 5); // 最多5条
```

### 3.7 风险提示生成

```typescript
const warnings: string[] = [];

if (name.includes('ST')) warnings.push('ST股票，风险较高');
if (peRatio > 100) warnings.push(`市盈率过高(${peRatio.toFixed(1)})`);
if (turnoverRate > 20) warnings.push('换手率过高，筹码不稳定');
```

## 4. 性能优化策略

### 4.1 异步执行

```typescript
// AI分析不影响主流程，失败不阻断
try {
  aiAnalysis = performAIAnalysis(klineData, tempOpportunityData);
} catch (error) {
  console.warn(`[${code}] AI分析失败:`, error);
  // 继续执行，不抛出异常
}
```

### 4.2 错误隔离

每个股票的 AI 分析独立进行，单个失败不影响其他股票。

### 4.3 数据缓存

分析结果自动保存到`StockOpportunityData.aiAnalysis`字段，随整体数据缓存到 IndexedDB。

### 4.4 计算优化

- 特征提取时只取最近 20 根 K 线
- 相似度计算使用向量化操作
- 提前终止：达到搜索上限即停止

## 8. 已知限制

1. **数据依赖**：需要至少 30 根 K 线才能有效预测
2. **技术指标局限**：纯技术分析，未考虑基本面
3. **市场环境影响**：未考虑大盘、政策等外部因素
4. **实时性**：基于历史数据，无法预测突发事件
