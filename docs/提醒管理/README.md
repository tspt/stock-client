## 📊 功能概览

### 核心功能

```
提醒管理功能体系
├── 价格提醒
│   ├── 目标价格设置
│   ├── 实时价格监控
│   └── 触发通知
│
└── 智能预警
    ├── 支撑/阻力位突破预警
    ├── 成交量异常放大预警
    ├── 技术指标金叉/死叉预警
    └── 自动检测和分析
```

### 技术特性

- ✅ **实时监控**：自动检测股票价格变化
- ✅ **智能分析**：支撑阻力位、成交量、技术指标多维度预警
- ✅ **即时通知**：触发时发送系统通知
- ✅ **灵活配置**：支持自定义预警参数
