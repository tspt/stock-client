# 提醒管理 - 使用指南

> 📚 **相关文档**: [技术实现](./02-技术实现.md) | [返回上级](./README.md)

本文档面向**普通用户**，介绍提醒管理页面的使用方法和预警配置。

---

## 🎯 功能概览

提醒管理页面提供价格提醒和智能预警功能，帮助用户实时监控股票动态。

### 核心能力

- ✅ **价格提醒**：目标价格涨到/跌到提醒
- ✅ **幅度提醒**：涨跌幅百分比提醒
- ✅ **支撑阻力位**：突破关键价位提醒
- ✅ **成交量异常**：成交量异动提醒
- ✅ **指标金叉/死叉**：MACD、KDJ、RSI、MA 等技术指标交叉提醒

---

## 📋 预警类型详解

### 1️⃣ 价格提醒（基础）

**触发条件**：

- 价格涨到指定价位
- 价格跌到指定价位

**使用场景**：

- 设置止盈/止损位
- 关注特定价格点位

### 2️⃣ 幅度提醒（基础）

**触发条件**：

- 涨幅达到指定百分比
- 跌幅达到指定百分比

**使用场景**：

- 监控大幅波动
- 捕捉快速涨跌

### 3️⃣ 支撑/阻力位突破（高级）✨

**功能说明**：

- 自动识别股票的支撑位和阻力位
- 当价格突破阻力位或跌破支撑位时发出提醒
- 基于局部高低点聚类算法，智能识别关键价位

**配置选项**：

- 选择"突破阻力位"或"跌破支撑位"
- 回溯周期（默认 60 天）
- 聚类容差参数

**使用场景**：

- 突破交易策略：捕捉价格突破关键阻力位的时机
- 止损设置：监控支撑位是否被跌破
- 技术分析辅助：自动识别重要的技术位

### 4️⃣ 成交量异常放大（高级）✨

**功能说明**：

- 实时监控成交量变化
- 当成交量超过历史均量的指定倍数时发出提醒
- 可自定义检测周期和倍数阈值

**配置选项**：

- 倍数阈值（如 2.0 倍）
- 检测周期（如 20 天）
- 异常类型：放量（spike）或缩量（shrink）

**使用场景**：

- 捕捉主力异动：大资金进场或离场
- 验证价格突破：放量突破更有说服力
- 发现潜在机会：异常放量往往预示重要行情

### 5️⃣ 技术指标金叉/死叉（高级）✨

**支持的指标**：

| 指标 | 全称               | 用途                     |
| ---- | ------------------ | ------------------------ |
| MACD | 平滑异同移动平均线 | 趋势跟踪                 |
| KDJ  | 随机指标           | 超买超卖判断             |
| RSI  | 相对强弱指标       | 动量分析                 |
| MA   | 移动平均线         | 趋势确认（可自定义周期） |

**配置选项**：

- 选择指标类型
- 选择"金叉"或"死叉"
- MA 指标可自定义快慢周期

**使用场景**：

- 趋势确认：金叉确认上涨趋势，死叉确认下跌趋势
- 买卖点提示：结合多个指标提高准确性
- 技术分析自动化：无需手动盯盘

## 🔧 使用方法

### 创建新的预警

**步骤 1**：进入设置页面

- 在股票列表或详情页点击"设置提醒"按钮

**步骤 2**：选择提醒类型

- 价格提醒 / 幅度提醒（基础）
- 支撑阻力位 / 成交量异常 / 指标交叉（高级）

**步骤 3**：配置触发条件

**支撑阻力位**：

- 选择"突破阻力位"或"跌破支撑位"
- 系统自动识别关键价位

**成交量异常**：

- 设置倍数阈值（如 2.0 倍）
- 设置检测周期（如 20 天）

**指标交叉**：

- 选择指标类型（MACD/KDJ/RSI/MA）
- 选择"金叉"或"死叉"
- MA 指标需设置快慢周期

**步骤 4**：设置其他参数

- 时间周期
- 通知方式（系统通知/声音提醒）

**步骤 5**：保存提醒

### 查看预警列表

在"智能预警"页面可以查看所有已设置的提醒，包括：

- 提醒类型（用不同颜色标签区分）
- 触发条件详细说明
- 当前价格和状态
- 时间周期和通知方式

## 💾 数据特性

### 自动检测

- **检测频率**：跟随行情数据更新频率
- **静默检测**：后台自动检测，不影响使用
- **即时通知**：满足条件立即发送通知

### 回退机制

所有提醒都实现了回退机制：

- 触发后需价格回退到另一侧才能再次触发
- 避免同一条件下反复提醒

## ⚠️ 注意事项

### K 线数据依赖

**重要**：以下高级预警需要 K 线数据：

- 支撑/阻力位突破
- 成交量异常
- 指标金叉/死叉

**现状**：

- 目前仅在有机会分析数据的股票上生效
- 后续可扩展为自动获取 K 线数据

### 性能优化

- 支撑阻力位识别使用聚类算法，计算量较大
- 建议合理设置回溯周期（默认 60 天）
- 避免同时设置过多的复杂提醒

### 准确性说明

- 支撑阻力位识别基于历史数据，仅供参考
- 成交量异常检测受市场整体环境影响
- 技术指标信号应结合其他分析方法使用

## 🆘 常见问题

### Q1: 为什么某些预警没有触发？

**A:**

1. 检查是否为高级预警（需要 K 线数据）
2. 确认该股票有机会分析数据
3. 检查预警是否处于启用状态
4. 查看触发条件是否正确配置

### Q2: 如何判断支撑/阻力位是否准确？

**A:**

1. 支持位/阻力位基于历史数据识别
2. 可结合 K 线图手动验证
3. 多个价位聚类更可靠
4. 仅供参考，需结合其他分析

### Q3: 成交量异常倍数设置多少合适？

**A:**

- 保守：1.5-2.0 倍
- 适中：2.0-3.0 倍
- 激进：3.0 倍以上
- 建议根据个股历史表现调整

### Q4: 哪些指标组合最有效？

**A:**
常用组合：

- MACD 金叉 + 成交量放大 → 强烈看涨
- KDJ 超卖 + RSI 背离 → 可能反弹
- MA 金叉 + 突破阻力位 → 趋势确认
