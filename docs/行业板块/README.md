## 📊 功能概览

### 核心功能

```
行业板块功能体系
├── 板块列表
│   ├── 约128个行业板块实时监控
│   ├── 完整的资金流向分析
│   ├── 多维度排序
│   └── 分页显示（每页50条）
│
├── 搜索和筛选
│   ├── 按板块名称、代码搜索
│   ├── 按领涨股名称搜索
│   ├── 升序/降序切换
│   └── 实时搜索结果更新
│
└── 行业详情查看
    ├── 点击板块名称查看成分股
    ├── 详细的资金流向数据
    ├── 成分股排序和筛选
    └── 抽屉式展示
```

### 技术特性

- ✅ **实时数据**：东方财富 API 数据源
- ✅ **自动刷新**：10 秒间隔轮询更新
- ✅ **成分股查看**：抽屉式交互，不影响主页面
- ✅ **错误处理**：完善的异常处理和重试机制
