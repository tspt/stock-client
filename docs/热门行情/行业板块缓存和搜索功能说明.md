# 行业板块缓存和搜索功能说明

## 功能概述

本次更新为行业板块页面添加了以下功能:

1. **所有行业分类缓存机制** - 每日只拉取一次,使用 localStorage 缓存
2. **智能搜索功能** - 基于本地缓存数据进行搜索
3. **视图切换** - 支持在行业列表和成分股详情之间切换
4. **动态轮询** - 根据当前视图自动切换轮询目标

## 技术实现

### 1. 类型定义 (`src/types/stock.ts`)

新增 `IndustrySectorBasicInfo` 接口:

```typescript
export interface IndustrySectorBasicInfo {

```

### 2. 服务层 (`src/services/hot/industry-sectors.ts`)

新增 `getAllIndustrySectors()` 函数:

- **缓存策略**: localStorage + 24 小时 TTL
- **并发控制**: 使用 Promise 缓存避免重复请求
- **API 端点**: `/api/eastmoney-data/bkzj/getbkzj?key=f62&code=m%3A90%2Bs%3A4`
- **代理配置**: 通过 Vite 代理转发到 `https://data.eastmoney.com/dataapi/bkzj/getbkzj`,自动携带 Cookie
- **返回字段**: f12(代码), f14(名称), f62(主力净流入)


## 用户操作流程

### 场景 1: 浏览行业列表

1. 打开行业板块页面
2. 默认显示行业列表(每页 50 条)
3. 每 10 秒自动刷新最新数据
4. 可排序、翻页、手动刷新

### 场景 2: 搜索并查看某行业成分股

1. 在搜索框输入行业名称或代码
2. AutoComplete 显示匹配的行业列表
3. 点击某个行业
4. 页面自动切换到该行业的成分股列表
5. 顶部显示"当前查看: XXX 行业"和"返回列表"按钮
6. 表格标题变为"{行业名称} - 成分股列表"
7. 每 10 秒自动刷新该行业的成分股数据

### 场景 3: 查看行业详情(抽屉)

1. 在行业列表中点击某个行业名称
2. 右侧弹出抽屉,显示该行业的成分股
3. 抽屉关闭后不影响主视图状态

### 场景 4: 返回行业列表

1. 在成分股详情模式下
2. 点击顶部的"返回列表"按钮
3. 页面切回行业列表视图
4. 重新加载行业列表数据

## 缓存机制

### 缓存键

```typescript
const ALL_INDUSTRY_SECTORS_CACHE_KEY = 'all_industry_sectors_v1';
```

### 缓存结构

```typescript
interface AllIndustrySectorsCache {
  savedAt: number; // 缓存时间戳
  sectors: IndustrySectorBasicInfo[]; // 行业数据
}
```

### 过期策略

- TTL: 24 小时 (86400000ms)
- 检查条件: `Date.now() - savedAt < TTL`
- 过期后自动重新拉取

### 并发控制

```typescript
let allIndustrySectorsFetchPromise: Promise<IndustrySectorBasicInfo[]> | null = null;

// 如果已有正在进行的请求,直接返回该Promise
if (allIndustrySectorsFetchPromise) {
  return allIndustrySectorsFetchPromise;
}
```

## API 说明

### 获取所有行业分类

```
GET /api/eastmoney-data/bkzj/getbkzj?key=f62&code=m%3A90%2Bs%3A4
```

**代理配置**:

- Vite 代理: `/api/eastmoney-data` → `https://data.eastmoney.com/dataapi`
- Cookie: 在 `vite.config.ts` 中配置,自动携带

**响应字段:**

- f12: 板块代码 (如 "BK0459")
- f13: 市场类型 (90)
- f14: 板块名称 (如 "元件")
- f62: 主力净流入 (元)

### 获取行业板块列表(分页)

```
GET /api/eastmoney/clist/get?fs=m%3A90+s%3A4&fields=...&pn=1&pz=50&fid=f3&po=1
```

### 获取行业成分股(分页)

```
GET /api/eastmoney/clist/get?fs=b:BK1020&fields=...&pn=1&pz=50&fid=f3&po=1
```

## 注意事项

1. **表格列保持不变**: 无论是行业列表还是成分股列表,都使用相同的列定义(净额+净占比两列布局)

2. **抽屉与主视图独立**:

   - `drawerSector`: 抽屉中查看的行业
   - `selectedSector`: 主视图中选中的行业(成分股模式)
   - 两者互不干扰

3. **搜索仅用于跳转**: 搜索不会过滤当前表格数据,而是用于快速定位并跳转到某个行业的成分股

4. **轮询频率**: 固定为 10 秒,根据视图模式自动切换轮询目标

5. **缓存清理**: 如需强制刷新缓存,可在浏览器控制台执行:
   ```javascript
   localStorage.removeItem('all_industry_sectors_v1');
   location.reload();
   ```
