## 📊 功能概览

### 核心功能

```
热门行情
├── 热门板块
│   ├── 按涨跌幅排序
│   ├── 按成交量排序
│   └── 按成交额排序
│
├── 热门股票
│   ├── 按涨跌幅排序
│   ├── 按成交量排序
│   ├── 按成交额排序
│   └── 按换手率排序
│
├── 热门概念
│   └── 当前市场热门题材
│
├── 资金动向
│   └── 主力资金流向分析
│
└── 市场情绪
    └── 整体市场情绪指标
```

### 技术特性

- ✅ **多维度排序**：支持多种排序方式
- ✅ **实时数据**：可配置轮询更新
- ✅ **API 代理**：解决跨域问题
- ✅ **模拟数据**：开发阶段可用
