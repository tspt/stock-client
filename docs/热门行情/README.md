## 📊 功能概览

### 核心功能

```
热门行情功能体系
├── 指数展示
│   ├── 主要市场指数实时监控
│   ├── 响应式卡片布局
│   └── 自动刷新（10秒间隔）
│
└── 板块排行
    ├── 领涨概念板块TOP20
    ├── 领跌概念板块TOP20
    ├── 资金流向数据
    └── 上涨/下跌家数统计
```

### 技术特性

- ✅ **实时数据**：东方财富 API 数据源
- ✅ **自动刷新**：10 秒间隔轮询更新
- ✅ **响应式布局**：适配不同屏幕尺寸
- ✅ **错误处理**：完善的异常处理和重试机制
