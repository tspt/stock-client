# 热门行情功能实现说明

## 功能概述

在热门行情页面实现了指数展示和板块排行功能，包括东方财富的领涨/领跌概念板块展示。页面采用响应式布局，顶部显示主要市场指数，下方展示领涨和领跌的概念板块。

## 实现内容

### 1. 类型定义 (`src/types/stock.ts`)

新增了多个接口类型:

- `EastMoneySectorData`: 东方财富热门板块数据

  - code: 板块代码
  - name: 板块名称
  - changePercent: 涨跌幅（百分比）
  - turnoverRate: 换手率（百分比）
  - riseCount: 上涨家数
  - fallCount: 下跌家数
  - leadingStockName: 领涨股票名称
  - leadingStockCode: 领涨股票代码
  - leadingStockChangePercent: 领涨股票涨跌幅（百分比）

- `EastMoneyIndexData`: 东方财富指数数据
  - code: 指数代码
  - name: 指数名称
  - value: 当前点位
  - change: 涨跌额
  - changePercent: 涨跌幅（百分比）
  - volume: 成交量
  - amount: 成交额

### 2. API 服务 (`src/services/hot/eastmoney-sectors.ts` 和 `src/services/hot/indices.ts`)

实现了多个 API 函数:

- `getEastMoneyRisingSectors(count)`: 获取东方财富领涨概念板块
- `getEastMoneyFallingSectors(count)`: 获取东方财富领跌概念板块
- `getEastMoneySectorRanks(count)`: 同时获取领涨和领跌概念板块
- `getEastMoneyIndices()`: 获取东方财富主要市场指数

API 接口地址:

- 板块排行: `/api/eastmoney/clist/get`
- 指数数据: `/api/eastmoney/qt/index/list`

### 3. 组件实现 (`src/components/EastMoneySectorRankCard/`)

创建了 `EastMoneySectorRankCard` 组件用于展示板块排行数据:

**Props:**

- title: 卡片标题
- data: 板块数据数组
- type: 'rising' | 'falling' (用于控制颜色显示)

**表格列:**

- 排名: 显示序号(带圆形背景)
- 板块名称: 板块名称(支持省略号)
- 涨跌幅: 百分比显示(领涨红色,领跌绿色)
- 上涨/下跌家数: 显示板块内上涨和下跌的股票数量
- 领涨股: 显示领涨股票名称和涨跌幅

### 4. 指数卡片组件 (`src/pages/HotPage/components/IndexCard.tsx`)

创建了 `IndexCard` 组件用于展示单个指数信息:

**Props:**

- index: 指数数据对象

**显示内容:**

- 指数名称
- 当前点位
- 涨跌额和涨跌幅
- 成交量和成交额

### 5. 状态管理 (`src/stores/hotStore.ts`)

在 hotStore 中添加了:

**State:**

- indices: 指数数据列表
- indicesLoading: 指数数据加载状态
- eastMoneyRisingSectors: 东方财富领涨概念板块
- eastMoneyFallingSectors: 东方财富领跌概念板块
- sectorsLoading: 板块数据加载状态

**Actions:**

- loadEastMoneyIndices(): 加载东方财富指数数据
- loadEastMoneySectorRanks(): 加载东方财富板块排行数据

### 6. 页面集成 (`src/pages/HotPage/HotPage.tsx`)

在 HotPage 中:

1. 导入 IndexCard 和 EastMoneySectorRankCard 组件
2. 从 store 获取指数和板块数据及加载状态
3. 使用 usePolling hook 实现定时刷新(10 秒间隔)
4. 在页面顶部展示指数卡片网格
5. 在下方展示领涨和领跌概念板块卡片

## 布局规范

按照项目规范:

- ✅ 指数展示区域固定在顶部
- ✅ 板块排行区域可滚动
- ✅ 左右两列布局展示领涨和领跌板块
- ✅ 响应式设计适配不同屏幕尺寸
- ✅ 统一的色彩方案和样式

## 使用说明

1. 启动应用后，进入"热门行情"页面
2. 页面顶部会自动加载并显示主要市场指数
3. 页面下方会显示领涨概念板块和领跌概念板块
4. 每个板块列表显示前 20 个板块
5. 数据每 10 秒自动刷新一次
6. 点击表头可以排序
7. 表格支持垂直滚动查看所有板块

## 数据更新

- 指数数据和板块数据每 10 秒自动刷新
- 使用 usePolling hook 实现定时轮询
- 支持手动刷新按钮
- 错误处理和重试机制
