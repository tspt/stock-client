# 领涨/领跌板块功能实现说明

## 功能概述

在数据概况页面顶部实现了领涨板块和领跌板块的展示功能,采用固定宽度布局(300px),从左往右排列。

## 实现内容

### 1. 类型定义 (`src/types/stock.ts`)

新增了两个接口类型:

- `LeadingStock`: 领涨/领跌股票信息

  - code: 股票代码
  - name: 股票名称
  - change: 涨跌额
  - changePercent: 涨跌幅
  - currentPrice: 最新价

- `SectorRankData`: 板块排行数据
  - code: 板块代码
  - name: 板块名称
  - changePercent: 涨跌幅
  - change: 涨跌额
  - leadingStock: 领涨/领跌股信息
  - turnoverRate: 换手率
  - volumeRatio: 量比
  - circulatingMarketCap: 流通市值(亿)
  - marketCap: 总市值(亿)
  - volume: 成交量(万手)
  - amount: 成交额(万)
  - changePercent5d/20d/60d/52w/YTD: 各周期涨跌幅

### 2. API 服务 (`src/services/sectorApi.ts`)

实现了三个 API 函数:

- `getRisingSectors(count)`: 获取领涨板块数据
- `getFallingSectors(count)`: 获取领跌板块数据
- `getSectorRanks(count)`: 同时获取领涨和领跌板块数据

API 接口地址:

- 领涨: `https://proxy.finance.qq.com/cgi/cgi-bin/rank/pt/getRank?board_type=hy2&sort_type=priceRatio&direct=down&offset=0&count=20`
- 领跌: `https://proxy.finance.qq.com/cgi/cgi-bin/rank/pt/getRank?board_type=hy2&sort_type=priceRatio&direct=up&offset=0&count=20`

### 3. 组件实现 (`src/components/SectorRankCard/`)

创建了 `SectorRankCard` 组件用于展示板块排行数据:

**Props:**

- title: 卡片标题
- data: 板块数据数组
- loading: 加载状态
- type: 'rising' | 'falling' (用于控制颜色显示)

**表格列:**

- 排名: 显示序号(带圆形背景)
- 板块名称: 板块名称(支持省略号)
- 涨跌幅: 百分比显示(领涨红色,领跌绿色)
- 领涨股: 显示领涨股票名称和涨跌幅

### 4. 状态管理 (`src/stores/overviewStore.ts`)

在 overviewStore 中添加了:

**State:**

- risingSectors: 领涨板块数据
- fallingSectors: 领跌板块数据
- sectorsLoading: 板块数据加载状态

**Action:**

- loadSectorRanks(): 加载板块排行数据

### 5. 页面集成 (`src/pages/OverviewPage/OverviewPage.tsx`)

在 OverviewPage 中:

1. 导入 SectorRankCard 组件
2. 从 store 获取板块数据和加载状态
3. 在 useEffect 中调用 loadSectorRanks() 加载数据
4. 在 Content 顶部添加板块展示区域

### 7. 代理配置

**Vite 开发环境 (`vite.config.ts`):**

```javascript
'/api/tencent/rank': {
  target: 'https://proxy.finance.qq.com',
  changeOrigin: true,
  rewrite: (path) => path.replace(/^\/api\/tencent\/rank/, '/cgi/cgi-bin/rank'),
  headers: {
    Referer: 'https://finance.qq.com',
    Origin: 'https://finance.qq.com',
  },
}
```

**Electron 生产环境 (`electron/localApiProxy.ts`):**

- 添加了 `/api/tencent/rank` 代理配置
- 特殊处理路径转换: `/api/tencent/rank` → `/cgi/cgi-bin/rank`

## 布局规范

按照项目规范:

- ✅ 使用固定宽度布局(300px)
- ✅ 从左往右排列(领涨板块在左,领跌板块在右)
- ✅ 两列之间有 16px 间距
- ✅ 底部有边框分隔线

## 使用说明

1. 启动应用后,进入"数据概况"页面
2. 页面顶部会自动加载并显示领涨板块和领跌板块
3. 每个板块列表显示前 20 个板块
4. 点击表头可以排序
5. 表格支持垂直滚动查看所有板块

## 数据更新

目前板块数据在页面加载时获取一次,如需定时刷新,可以在组件中添加定时器或手动刷新按钮。
