# 热门行情 - 使用指南

> 📚 **相关文档**: [技术实现](./02-技术实现.md) | [返回上级](./README.md)

本文档面向**普通用户**，介绍热门行情页面的使用方法和数据解读。

---

## 🎯 功能概览

热门行情页面提供实时市场指数展示和板块排行分析，帮助投资者快速把握市场热点。

### 核心能力

- ✅ **指数监控**：主要市场指数实时展示
- ✅ **板块排行**：领涨/领跌概念板块 TOP20
- ✅ **自动刷新**：10 秒间隔自动更新数据
- ✅ **资金流向**：完整的板块资金数据分析

---

## 📋 页面布局

### 1️⃣ 指数展示区域（顶部固定）

**位置**：页面顶部，始终可见

**显示内容**：

- 上证指数、深证成指、创业板指、科创 50、沪深 300 等主要指数
- 每个指数卡片显示：
  - 指数名称
  - 当前点位
  - 涨跌额和涨跌幅
  - 成交量和成交额

**颜色标识**：

- 🔴 红色 = 上涨
- 🟢 绿色 = 下跌

### 2️⃣ 板块排行区域（可滚动）

**位置**：指数展示区域下方

**布局**：左右两列

- **左列**：领涨概念板块（涨幅最大的前 20 个）
- **右列**：领跌概念板块（跌幅最大的前 20 个）

**特性**：

- 支持垂直滚动查看所有板块
- 每 10 秒自动刷新数据
- 显示板块内上涨/下跌家数统计

## 📊 数据解读

### 指数卡片解读

**关注要点**：

1. **涨跌幅度**：反映市场整体走势

   - 大幅上涨 → 市场情绪乐观
   - 大幅下跌 → 市场情绪悲观

2. **成交量和成交额**：反映市场活跃度

   - 放量上涨 → 趋势可能持续
   - 缩量下跌 → 调整可能性大

3. **对比分析**：不同指数表现差异
   - 上证强于创业板 → 蓝筹股占优
   - 创业板强于上证 → 成长股活跃

### 板块排行解读

**领涨板块**：

- 反映当前市场热点
- 关注持续性强的概念
- 结合领涨股分析板块强度

**领跌板块**：

- 提示风险区域
- 避免追高已大幅上涨的板块
- 关注板块轮动规律

**关键指标**：

| 指标          | 含义           | 解读                     |
| ------------- | -------------- | ------------------------ |
| 涨跌幅        | 板块整体涨跌   | >3%为强势，<-3%为弱势    |
| 换手率        | 交易活跃程度   | 高换手=资金活跃          |
| 上涨/下跌家数 | 板块内个股表现 | 上涨多=普涨，少数涨=分化 |
| 领涨股        | 板块龙头       | 龙头强弱影响板块持续性   |

## 🔧 使用技巧

### 推荐观察组合

#### 🎯 热点追踪模式

1. 关注领涨板块 TOP5
2. 查看领涨股的涨跌幅
3. 观察板块内上涨家数占比
4. 判断热点持续性

#### 🛡️ 风险控制模式

1. 关注领跌板块 TOP5
2. 避免持有领跌板块股票
3. 观察是否有资金流出迹象
4. 及时调整持仓结构

#### 📈 趋势判断模式

1. 对比主要指数表现
2. 观察成交量变化
3. 分析板块轮动规律
4. 判断市场整体趋势

## 💾 数据特性

### 自动刷新

- **刷新频率**：每 10 秒自动更新
- **首次加载**：打开页面立即获取数据
- **后台刷新**：不影响页面浏览体验

### 数据来源

- **数据提供商**：东方财富网
- **数据准确性**：实时行情数据
- **更新延迟**：<1 秒

## ⚠️ 注意事项

### 数据要求

- 需要网络连接才能获取实时数据
- 某些板块可能缺少部分数据字段（显示为"-"），这是正常现象

### 性能优化

- 首次加载可能需要几秒钟
- 数据量大时建议耐心等待
- 如遇卡顿可尝试刷新页面

## 🆘 常见问题

### Q1: 数据不更新怎么办？

**A:**

1. 检查网络连接是否正常
2. 尝试手动刷新页面
3. 查看浏览器控制台是否有错误信息
4. 如问题持续，联系技术支持

### Q2: 为什么有些数据显示为"-"？

**A:** 某些板块可能缺少特定数据字段，这是正常现象，不影响其他数据的准确性。

### Q3: 页面加载缓慢怎么办？

**A:**

1. 检查网络状况
2. 关闭不必要的浏览器标签页
3. 清除浏览器缓存后重试
4. 如问题持续，可能是服务器繁忙，稍后再试

### Q4: 如何判断板块热点是否可持续？

**A:**

1. 观察领涨股的持续性
2. 关注板块内上涨家数占比
3. 查看成交量是否放大
4. 结合消息面和政策面分析
