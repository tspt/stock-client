# 热门行情 - 使用指南

> 📚 **相关文档**: [技术实现](./02-技术实现.md) | [返回上级](./README.md)

本文档介绍热门行情页面的功能和使用方法。

---

## 🎯 功能概览

热门行情页面展示当前市场中最活跃的内容，帮助您快速把握市场脉搏。

### 核心模块

#### 1️⃣ 热门板块

展示行业板块的实时表现，支持多种排序方式：

| 排序方式 | 说明               | 适用场景          |
| -------- | ------------------ | ----------------- |
| 涨跌幅   | 按板块指数涨跌排序 | 发现强势/弱势板块 |
| 成交量   | 按成交总量排序     | 发现活跃板块      |
| 成交额   | 按成交金额排序     | 发现资金关注板块  |

**用途**：

- 识别当前市场的热点行业
- 发现资金流入流出的板块
- 辅助个股选择（选强板块中的强势股）

---

#### 2️⃣ 热门股票

展示个股的实时表现，支持四种排序：

| 排序方式 | 说明           | 适用场景        |
| -------- | -------------- | --------------- |
| 涨跌幅   | 按股价涨跌排序 | 发现涨停/跌停股 |
| 成交量   | 按成交手数排序 | 发现高换手股    |
| 成交额   | 按成交金额排序 | 发现大资金股    |
| 换手率   | 按换手率排序   | 发现活跃交易股  |

**用途**：

- 捕捉市场热点个股
- 发现异常波动的股票
- 跟踪主力资金动向

---

#### 3️⃣ 热门概念

展示当前市场热门的概念题材，如：

- 人工智能
- 新能源汽车
- 半导体芯片
- 医药生物
- ...

**用途**：

- 了解市场炒作主线
- 发现题材轮动规律
- 辅助主题投资

---

#### 4️⃣ 资金动向

分析主力资金的流向情况：

| 指标         | 说明                    |
| ------------ | ----------------------- |
| 主力净流入   | 超大单 + 大单的净买入额 |
| 超大单净流入 | 机构级别资金流向        |
| 大单净流入   | 大户资金流向            |

**用途**：

- 判断主力资金态度
- 发现资金持续流入的标的
- 辅助买卖决策

---

#### 5️⃣ 市场情绪

综合评估整体市场的情绪状态：

| 情绪指标 | 说明                        |
| -------- | --------------------------- |
| 上涨家数 | 今日上涨的股票数量          |
| 下跌家数 | 今日下跌的股票数量          |
| 涨停家数 | 今日涨停的股票数量          |
| 跌停家数 | 今日跌停的股票数量          |
| 情绪指数 | 综合计算的情绪得分（0-100） |

**情绪分级**：

- **80-100**：极度乐观，注意风险
- **60-80**：偏乐观，可适当参与
- **40-60**：中性，观望为主
- **20-40**：偏悲观，谨慎操作
- **0-20**：极度悲观，可能见底

---

## 🔧 使用方法

### 查看热门板块

1. 打开热门行情页面
2. 点击"热门板块"标签
3. 选择排序方式（涨跌幅/成交量/成交额）
4. 查看排名前 50 的板块

**技巧**：

- 结合多个排序方式，全面分析
- 关注连续多日上榜的板块
- 注意板块内部的个股分化

---

### 查看热门股票

1. 点击"热门股票"标签
2. 选择排序方式
3. 设置显示数量（默认 50 只）
4. 查看排名列表

**技巧**：

- 涨跌幅榜：关注涨停股和跌幅榜
- 成交量榜：发现放量异动股
- 换手率榜：识别高活跃度股票
- 结合 K 线图分析走势

---

### 查看热门概念

1. 点击"热门概念"标签
2. 查看概念热度排名
3. 点击概念名称查看成分股

**技巧**：

- 关注新出现的热门概念
- 跟踪概念的持续性
- 分析概念的龙头股

---

### 查看资金动向

1. 点击"资金动向"标签
2. 选择排序方式（主力净流入/超大单/大单）
3. 查看资金流向排名

**技巧**：

- 重点关注主力持续净流入的标的
- 警惕主力大幅流出的股票
- 结合价格走势判断资金意图

---

### 查看市场情绪

1. 在页面顶部查看市场情绪卡片
2. 关注情绪指数变化
3. 对比上涨/下跌家数

**技巧**：

- 情绪指数 > 80：市场过热，注意回调风险
- 情绪指数 < 20：市场过冷，可能见底
- 结合成交量判断情绪真实性

---

## 💡 使用技巧

### 1. 多维度验证

不要只看单一指标，要综合分析：

- 板块热度 + 个股表现
- 资金流向 + 价格走势
- 市场情绪 + 成交量

### 2. 关注持续性

- 单日异动可能是偶然
- 连续多日上榜更有参考价值
- 观察热度的变化趋势

### 3. 结合基本面

热门行情只是技术面参考，还需结合：

- 公司基本面
- 行业发展前景
- 政策面影响

### 4. 注意风险

- 热门股波动大，注意止损
- 避免追高已大幅上涨的标的
- 控制仓位，分散投资

## ⚠️ 注意事项

### 数据延迟

- 数据可能有 3-5 分钟延迟
- 盘中数据为实时估算
- 收盘后数据更准确

### 模拟数据

**当前状态**：页面使用模拟数据

**原因**：

- 真实 API 需要接入第三方数据源
- 需要解决跨域问题
- 可能需要付费订阅

**后续计划**：

- 接入东方财富/同花顺 API
- 实现 WebSocket 实时更新
- 添加自定义提醒功能

## 🆘 常见问题

### Q1: 为什么数据显示的是模拟数据？

**原因**：

- 真实股票数据需要接入专业 API
- 目前处于开发阶段，使用模拟数据测试功能

**解决方案**：

- 查看 [02-技术实现](./02-技术实现.md) 了解如何接入真实 API
- 推荐数据源：东方财富、同花顺、新浪财经

---

### Q2: 数据多久更新一次？

**当前**：手动刷新或定时轮询（可配置）

**建议**：

- 盘中：每 30-60 秒更新
- 盘后：每日更新一次

---

### Q3: 如何导出热门股票列表？

**当前功能**：暂不支持导出

**计划功能**：

- 导出 Excel
- 导出 CSV
- 生成图片分享

---

### Q4: 可以自定义筛选条件吗？

**当前功能**：仅支持排序

**计划功能**：

- 价格范围筛选
- 市值范围筛选
- 涨跌幅阈值
- 自定义组合条件
