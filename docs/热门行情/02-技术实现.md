## 🏗️ 整体架构

### 数据流图

```
用户访问HotPage
    ↓
usePolling Hook (10秒轮询)
    ↓
hotStore Actions
    ├── loadEastMoneyIndices() → 获取指数数据
    └── loadEastMoneySectorRanks() → 获取板块排行
    ↓
API Services (src/services/hot/)
    ├── eastmoney-sectors.ts → 板块数据服务
    └── indices.ts → 指数数据服务
    ↓
东方财富 API (/api/eastmoney/clist/get)
    ↓
数据解析 & 状态更新
    ↓
UI 组件渲染
    ├── IndexCard (指数卡片)
    └── EastMoneySectorRankCard (板块排行卡片)
```

### 核心组件

| 组件             | 文件路径                                                             | 职责                         |
| ---------------- | -------------------------------------------------------------------- | ---------------------------- |

### 1. 数据服务层

**文件**: `src/services/hot/eastmoney-sectors.ts`

**核心函数**:

```typescript
// 获取东方财富领涨概念板块
export async function getEastMoneyRisingSectors(
  count: number = 20
): Promise<EastMoneySectorData[]> {
  const response = await fetch('/api/eastmoney/clist/get', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      fs: 'm:90 t:3 f:!50', // 概念板块过滤
      fid: 'f3', // 按涨跌幅排序
      po: '1', // 降序（领涨）
      pz: count.toString(), // 每页数量
      pn: '1', // 页码
      fields:
        'f12,f13,f14,f1,f2,f4,f3,f152,f20,f8,f104,f105,f128,f140,f141,f207,f208,f209,f136,f222',
    }),
  });

  const data = await response.json();
  return parseEastMoneySectorData(data.data.diff);
}

// 获取东方财富领跌概念板块
export async function getEastMoneyFallingSectors(
  count: number = 20
): Promise<EastMoneySectorData[]> {
  // 类似实现，po: '0' 表示升序
}

// 同时获取领涨和领跌概念板块
export async function getEastMoneySectorRanks(count: number = 20): Promise<{
  rising: EastMoneySectorData[];
  falling: EastMoneySectorData[];
}> {
  const [rising, falling] = await Promise.all([
    getEastMoneyRisingSectors(count),
    getEastMoneyFallingSectors(count),
  ]);
  return { rising, falling };
}
```

### 2. 轮询机制

**Hook**: `src/hooks/usePolling.ts`

```typescript
export function usePolling(
  callback: () => void,
  interval: number = 10000, // 默认10秒
  immediate: boolean = true // 首次立即执行
) {
  useEffect(() => {
    if (immediate) {
      callback();
    }

    const timer = setInterval(callback, interval);
    return () => clearInterval(timer);
  }, [callback, interval, immediate]);
}
```

**使用示例**:

```typescript
// HotPage.tsx
usePolling(() => {
  loadEastMoneyIndices();
  loadEastMoneySectorRanks();
}, 10000);
```

### 2. 请求参数详解

**板块排行参数**:

```javascript
{
  fs: 'm:90 t:3 f:!50',  // 概念板块过滤条件
  fid: 'f3',             // 排序字段：f3=涨跌幅
  po: '1',               // 排序方向：1=降序(领涨), 0=升序(领跌)
  pz: '20',              // 每页数量
  pn: '1',               // 页码
  fields: 'f12,f13,f14,f1,f2,f4,f3,f152,f20,f8,f104,f105,f128,f140,f141,f207,f208,f209,f136,f222'
  // 字段说明：
  // f12=代码, f13=交易所, f14=名称, f1=最新价, f2=涨跌额
  // f3=涨跌幅, f152=换手率, f20=成交量, f8=成交额
  // f104=上涨家数, f105=下跌家数, f128=领涨股名称
  // f140=领涨股代码, f141=领涨股涨跌幅
}
```

**指数数据参数**:

```javascript
{
  fs: 'm:1+s:2',         // 主要市场指数过滤
  fields: 'f12,f14,f2,f3,f4,f5,f6'
  // f12=代码, f14=名称, f2=最新价, f3=涨跌幅
  // f4=涨跌额, f5=成交量, f6=成交额
}
```
