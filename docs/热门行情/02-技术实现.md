### 步骤 2：处理跨域问题

#### 方案 A：Electron 主进程代理（推荐）✅

在 `electron/main.ts` 中添加请求拦截：

```typescript
import { session } from 'electron';

app.whenReady().then(() => {
  session.defaultSession.webRequest.onBeforeSendHeaders((details, callback) => {
    if (details.url.includes('sinajs.cn') || details.url.includes('gtimg.cn')) {
      details.requestHeaders['Referer'] = 'https://finance.sina.com.cn';
      details.requestHeaders['Origin'] = 'https://finance.sina.com.cn';
    }
    callback({ requestHeaders: details.requestHeaders });
  });
});
```

**优点**：

- ✅ 无需额外服务器
- ✅ 性能好
- ✅ 适合生产环境

---

#### 方案 B：本地 Node.js 代理

创建 `server/hotProxy.js`：

```javascript
const express = require('express');
const { createProxyMiddleware } = require('http-proxy-middleware');
const cors = require('cors');

const app = express();
app.use(cors());

// 代理东方财富 API
app.use(
  '/api/eastmoney',
  createProxyMiddleware({
    target: 'https://push2.eastmoney.com',
    changeOrigin: true,
    pathRewrite: { '^/api/eastmoney': '' },
  })
);

const PORT = 3001;
app.listen(PORT, () => {
  console.log(`Hot proxy server running on http://localhost:${PORT}`);
});
```

**优点**：

- ✅ 完全控制
- ✅ 可添加缓存
- ✅ 易于调试

**缺点**：

- ⚠️ 需要额外进程
- ⚠️ 增加复杂度

### 步骤 5：配置轮询更新

在 `src/stores/hotStore.ts` 中添加自动刷新：

```typescript
import { create } from 'zustand';

interface HotStore {
  hotSectors: HotSector[];
  hotStocks: HotStock[];
  loading: boolean;
  fetchHotSectors: (sortBy?: string) => Promise<void>;
  fetchHotStocks: (sortType?: string) => Promise<void>;
  startAutoRefresh: () => void;
  stopAutoRefresh: () => void;
}

export const useHotStore = create<HotStore>((set, get) => ({
  hotSectors: [],
  hotStocks: [],
  loading: false,

  fetchHotSectors: async (sortBy = 'changePercent') => {
    set({ loading: true });
    try {
      const sectors = await getHotSectors(sortBy);
      set({ hotSectors: sectors, loading: false });
    } catch (error) {
      set({ loading: false });
      console.error(error);
    }
  },

  startAutoRefresh: () => {
    // 每 60 秒刷新一次
    const interval = setInterval(() => {
      get().fetchHotSectors();
      get().fetchHotStocks();
    }, 60000);

    // 保存 interval ID 以便停止
    (get() as any).refreshInterval = interval;
  },

  stopAutoRefresh: () => {
    clearInterval((get() as any).refreshInterval);
  },
}));
```

### API 限流

- 注意 API 的调用频率限制
- 添加请求缓存机制
- 实现指数退避重试

```typescript
async function fetchWithRetry(url: string, retries = 3) {
  for (let i = 0; i < retries; i++) {
    try {
      return await axios.get(url);
    } catch (error) {
      if (i === retries - 1) throw error;
      await sleep(Math.pow(2, i) * 1000); // 指数退避
    }
  }
}
```
