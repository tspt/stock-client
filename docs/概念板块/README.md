# 概念板块功能使用指南

## 快速开始

### 1. 启动应用

```bash
npm run dev
```

### 2. 访问概念板块页面

在应用左侧导航栏中，找到并点击"概念板块"标签（位于"行业板块"下方）。

## 主要功能

### 概念板块列表

- **搜索**: 在顶部搜索框输入板块名称或代码进行筛选
- **排序**: 选择排序字段（涨跌幅、换手率、成交额、总市值）和排序方向（升序/降序）
- **分页**: 调整每页显示数量（20/50/100 条）
- **刷新**: 点击刷新按钮手动更新数据

### 热门概念板块

在"热门行情"页面中，可以看到：

- **领涨概念板块**: 涨幅最大的前 20 个概念板块
- **领跌概念板块**: 跌幅最大的前 20 个概念板块

数据每 10 秒自动更新一次。

## 数据说明

所有数据来源于东方财富网，包括：

- 实时价格信息
- 涨跌幅统计
- 成交量和成交额
- 市值数据
- 换手率和量比等技术指标

**注意**: 东方财富 API 需要有效的 Cookie 才能访问，系统已自动配置必要的 Cookie 信息。

## 常见问题

**Q: 为什么有些数据显示为"-"？**
A: 部分概念板块可能缺少某些数据字段，这是正常现象。

**Q: 数据更新频率是多少？**
A: 热门行情页面的概念板块数据每 10 秒更新一次，概念板块列表页面需要手动刷新。

**Q: 如何查看特定概念板块的详细信息？**
A: 目前仅支持列表查看，后续版本将添加详情页面。

## 技术支持

如遇问题，请检查：

1. 网络连接是否正常
2. 应用是否成功启动
3. 浏览器控制台是否有错误信息
