# 概念板块 - 使用指南

> 📚 **相关文档**: [技术实现](./02-技术实现.md) | [返回上级](./README.md)

本文档面向**普通用户**，介绍概念板块页面的使用方法和数据解读。

---

## 🎯 功能概览

概念板块页面提供 A 股市场概念板块的实时行情数据，包含详细的资金流向分析和成分股查看功能，帮助识别市场热点和投资机会。

### 核心能力

- ✅ **板块列表**：约 493 个概念板块实时监控
- ✅ **资金流向**：主力、超大单、大单、中单、小单五档分析
- ✅ **搜索筛选**：支持按名称、代码、领涨股搜索
- ✅ **成分股查看**：点击板块名称查看详细成分股

---

## 📋 页面布局

### 1️⃣ 顶部工具栏

**功能组件**：

- 🔍 **搜索框**：支持按板块名称、代码或领涨股名称搜索
- 🔄 **排序选择**：升序/降序切换
- 🔃 **刷新按钮**：手动刷新数据

**特性**：自适应不同屏幕尺寸

### 2️⃣ 数据表格区域

**表格特性**：

- 📌 **固定列**：板块名称列固定在左侧
- 📜 **可滚动**：横向和纵向均可滚动
- 📄 **分页控件**：底部显示分页信息（每页 50 条）
- ⏱️ **实时更新**：每 10 秒自动刷新当前页数据

---

## 📊 数据解读

### 显示字段说明

| 字段         | 含义              | 解读要点             |
| ------------ | ----------------- | -------------------- |
| 板块名称     | 概念板块名称      | 点击可查看成分股     |
| 今日涨跌幅   | 板块整体涨跌      | >3%强势，<-3%弱势    |
| 主力净流入   | 主力资金净额+占比 | 正数=流入，负数=流出 |
| 超大单净流入 | 超大资金净额+占比 | 机构资金动向         |
| 大单净流入   | 大资金净额+占比   | 大户资金动向         |
| 中单净流入   | 中等资金净额+占比 | 中户资金动向         |
| 小单净流入   | 散户资金净额+占比 | 散户资金动向         |

### 概念板块 vs 行业板块

**概念板块特点**：

- 数量更多（约 493 个 vs 128 个）
- 主题性强（如"人工智能"、"新能源汽车"）
- 波动性更大
- 更容易出现热点轮动

**行业板块特点**：

- 分类更稳定
- 基于产业链划分
- 波动相对平稳
- 更适合长期跟踪

## 🔧 使用技巧

### 推荐观察组合

#### 🎯 热点追踪模式

1. 按涨跌幅降序排列
2. 关注涨幅前 10 的概念板块
3. 查看主力净流入是否为正
4. 点击板块名称查看成分股表现
5. 判断热点持续性

#### 💰 资金追踪模式

1. 按主力净流入降序排列
2. 关注资金大幅流入的概念
3. 对比涨跌幅判断是否滞涨
4. 寻找资金提前布局的机会

#### 🛡️ 风险控制模式

1. 按涨跌幅升序排列
2. 关注跌幅前 10 的概念板块
3. 查看是否有资金流出
4. 避免持有领跌概念股票

## 🔍 搜索和筛选

### 搜索功能

**支持的搜索类型**：

- 概念名称模糊搜索（如"AI"、"新能源"）
- 板块代码精确搜索（如"BK0800"）
- 领涨股名称搜索（如"宁德时代"）

**使用技巧**：

- 输入关键词后实时显示结果
- 清空搜索框恢复完整列表
- 搜索结果保持排序状态

### 排序功能

**支持的排序方式**：

- 点击表头直接排序
- 使用排序选择器切换升序/降序
- 各资金流向列均可排序

**常用排序**：

- 涨跌幅 → 发现强弱概念
- 主力净流入 → 追踪机构动向
- 超大单净流入 → 观察大资金

---

## 📂 成分股查看

### 触发方式

点击任意板块名称，右侧弹出抽屉组件

### 抽屉内容

**显示信息**：

- 该概念的所有成分股列表
- 成分股的详细资金流向数据
- 支持对成分股进行排序和筛选

**操作方式**：

- 不影响主页面浏览体验
- 点击遮罩层或关闭按钮退出
- 可在抽屉内继续分析成分股

## 💾 数据特性

## ⚠️ 注意事项

### 数据要求

- 需要网络连接才能获取实时数据

### 性能优化

- 首次加载可能需要几秒钟
- 数据量大时建议耐心等待
- 如遇卡顿可尝试刷新页面

## 🆘 常见问题

### Q1: 数据加载失败怎么办？

**A:**

1. 检查网络连接是否正常
2. 尝试手动点击刷新按钮
3. 查看浏览器控制台是否有错误信息
4. 如问题持续，联系技术支持

### Q2: 搜索无结果怎么办？

**A:**

1. 确认搜索关键词拼写正确
2. 尝试简化搜索条件
3. 检查数据是否正常加载
4. 清空搜索框重试

### Q3: 如何判断概念热点是否可持续？

**A:**
综合以下指标：

1. 涨跌幅居前（>3%）
2. 主力净流入为正且较大
3. 超大单和大单同步流入
4. 领涨股表现强劲
5. 板块内多数股票上涨
6. 结合消息面和政策面分析
