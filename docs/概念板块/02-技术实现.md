## 🏗️ 整体架构

### 数据流图

```
用户访问 ConceptSectorPage
    ↓
useEffect (首次加载)
    ↓
调用 getUnifiedConceptBasic() - 加载所有概念基本信息（缓存）
    ↓
调用 getUnifiedConceptRank() - 加载当前页数据
    ↓
数据解析 & 格式化
    ↓
更新本地状态 (data, total, loading, allSectors)
    ↓
UI 组件渲染
    ├── AutoComplete 搜索框
    ├── 排序选择器
    ├── 数据表格
    └── 分页控件

用户输入搜索关键词
    ↓
防抖处理 (300ms)
    ↓
过滤 allSectors 显示匹配建议
    ↓
用户选择板块
    ↓
切换到详情模式 (viewMode = 'detail')
    ↓
调用 getSingleConceptSector() 获取详细数据
    ↓
显示单个板块信息，停止自动刷新

点击板块名称
    ↓
打开 ConceptSectorStocksDrawer
    ↓
调用 getConceptSectorStocks() API
    ↓
渲染成分股列表

usePolling Hook (每10秒)
    ↓
检查 viewMode === 'list'
    ↓
是：静默刷新当前页数据
    ↓
否：跳过刷新（详情模式）
```

```typescript
export async function getConceptSectors(
  sortType: string = 'f3',
  sortOrder: number = 1,
  pageSize: number = 50,
  pageNum: number = 1
): Promise<{ data: ConceptSectorRankData[]; total: number }> {
  const params = new URLSearchParams({
    cb: `jQuery${Date.now()}_${Math.random().toString().slice(2, 11)}`,
    fid: sortType,
    po: sortOrder.toString(),
    pz: pageSize.toString(),
    pn: pageNum.toString(),
    np: '1',
    fltt: '2',
    invt: '2',
    ut: '8dec03ba335b81bf4ebdf7b29ec27d15',
```

### 2. 数据解析

```typescript
function parseConceptSectorData(
  rawList: any[],
  total: number
): { data: ConceptSectorRankData[]; total: number } {
  const data: ConceptSectorRankData[] = rawList.map((item) => ({
    code: item.f12,
    name: item.f14,
    changePercent: item.f3,

    // 主力净流入（元转万元）
    mainNetInflow: item.f62 ? item.f62 / 10000 : undefined,
    mainNetInflowRatio: item.f184,

    // 超大单净流入（元转万元）
    superLargeNetInflow: item.f66 ? item.f66 / 10000 : undefined,
    superLargeNetInflowRatio: item.f69,

    // 大单净流入（元转万元）
    largeNetInflow: item.f72 ? item.f72 / 10000 : undefined,
    largeNetInflowRatio: item.f75,

    // 中单净流入（元转万元）
    mediumNetInflow: item.f78 ? item.f78 / 10000 : undefined,
    mediumNetInflowRatio: item.f81,

    // 小单净流入（元转万元）
    smallNetInflow: item.f84 ? item.f84 / 10000 : undefined,
    smallNetInflowRatio: item.f87,

    // 领涨股
    leadingStock: item.f204,
    leadingStockCode: item.f205,
  }));

  return { data, total };
}
```

```typescript
// 搜索选项（基于本地缓存数据）
const searchOptions = useMemo(() => {
  if (allSectors.length === 0) return [];

  // 如果没有搜索关键词，显示所有概念（最多50条）
```

### 6. 防重复请求机制

```typescript
// 防重复请求标记
const isLoadingRef = useRef(false);

// 加载数据（使用统一缓存服务）
const loadData = useCallback(
  async (page: number = 1, silent: boolean = false) => {
    if (isLoadingRef.current) return; // 如果正在加载，直接返回

    if (!silent) {
      setLoading(true);
    }
    isLoadingRef.current = true;

    try {
      const result = await getUnifiedConceptRank(sortOrder, pageSize, page);
      setData(result.data);
      setTotal(result.total);
      setCurrentPage(page);
    } catch (error) {
      logger.error('加载概念板块数据失败:', error);
      message.error('加载数据失败，请重试');
    } finally {
      if (!silent) {
        setLoading(false);
      }
      isLoadingRef.current = false;
    }
  },
  [sortOrder]
);
```

```typescript
  if (!value) return '-';

  if (absValue >= 10000) {
    // 大于等于1亿，显示为亿
```
