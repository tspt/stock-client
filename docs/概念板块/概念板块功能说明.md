# 概念板块功能说明

## 功能概述

新增了概念板块页面，用于展示和查询 A 股市场的概念板块行情数据。该功能基于东方财富 API 实现，提供了完整的概念板块数据展示和筛选功能。

## 功能特性

### 1. 概念板块列表页面

- **位置**: 在"行业板块"下方新增"概念板块"标签页
- **数据展示**: 显示概念板块的代码、名称、最新价、涨跌幅、换手率、量比、成交量、成交额、流通市值、总市值等信息
- **排序功能**: 支持按涨跌幅、换手率、成交额、总市值等字段排序
- **搜索功能**: 支持按板块名称或代码搜索
- **分页控制**: 支持 20/50/100 条数据显示

### 2. 热门行情页面增强

- 在热门行情页面的"领涨板块"和"领跌板块"下方，新增"领涨概念板块"和"领跌概念板块"展示
- 自动轮询更新概念板块数据（10 秒间隔）

### 3. 数据服务

- 提供完整的概念板块数据获取服务
- 支持多种排序方式和分页参数
- 数据解析和格式化

## 技术实现

### 1. API 接口

- 使用东方财富 API: `https://push2.eastmoney.com/api/qt/clist/get`
- 参数: `fs=m:90 t:3` 表示概念板块
- 支持 JSONP 格式响应

### 2. 文件结构

```
src/
├── services/
│   └── hot/
│       ├── concept-sectors.ts    # 概念板块服务
│       └── index.ts              # 导出概念板块相关函数
├── pages/
│   └── ConceptSectorPage/
│       ├── ConceptSectorPage.tsx     # 概念板块页面组件
│       └── ConceptSectorPage.module.css  # 样式文件
├── stores/
│   └── hotStore.ts               # 添加概念板块状态管理
└── App.tsx                       # 添加概念板块路由
```

### 3. 代理配置

在 `electron/localApiProxy.ts` 中添加了东方财富 API 的代理配置，并配置了必要的 Cookie：

```typescript
'/api/eastmoney': {
  target: 'https://push2.eastmoney.com',
  referer: 'https://data.eastmoney.com',
  origin: 'https://data.eastmoney.com',
}
```

**重要**: 东方财富 API 需要有效的 Cookie 才能访问，已在代理层和客户端请求中配置了必要的 Cookie 信息。

## 使用方法

### 1. 查看概念板块列表

1. 点击左侧导航栏的"概念板块"标签
2. 使用顶部的筛选器选择排序字段和方向
3. 使用搜索框查找特定概念板块
4. 调整每页显示数量

### 2. 查看热门概念板块

1. 在"热门行情"页面中查看
2. "领涨概念板块"显示涨幅最大的概念
3. "领跌概念板块"显示跌幅最大的概念

## 数据字段说明

| 字段     | 说明           | 单位 |
| -------- | -------------- | ---- |
| 代码     | 概念板块代码   | -    |
| 名称     | 概念板块名称   | -    |
| 最新价   | 当前价格       | 元   |
| 涨跌幅   | 价格变化百分比 | %    |
| 换手率   | 当日换手率     | %    |
| 量比     | 成交量比率     | -    |
| 成交量   | 当日成交量     | 万手 |
| 成交额   | 当日成交额     | 万元 |
| 流通市值 | 流通市值       | 亿元 |
| 总市值   | 总市值         | 亿元 |

## 注意事项

1. 概念板块数据来源于东方财富，可能存在延迟
2. 数据刷新频率为 10 秒一次
3. 部分字段可能为空值，显示为"-"
4. 涨跌幅颜色：红色表示上涨，绿色表示下跌
