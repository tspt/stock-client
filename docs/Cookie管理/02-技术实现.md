## 💾 数据存储

### IndexedDB 设计

#### Cookies 表结构

```typescript
interface CookieEntry {
  id: string; // 唯一标识（UUID）
  value: string; // Cookie 字符串
  createdAt: number; // 创建时间戳
  lastUsedAt: number; // 最后使用时间戳
  successCount: number; // 成功请求次数
  failureCount: number; // 失败请求次数
  isActive: boolean; // 是否激活
  healthScore: number; // 健康评分 (0-100)
}
```

#### 索引设计

```typescript
// 主键
keyPath: 'id'

// 索引
- isActive (用于快速查询活跃 Cookie)
- healthScore (用于排序)
- createdAt (用于清理旧数据)
```

### 数据库初始化

```typescript
const DB_NAME = 'CookiePoolDB';
const DB_VERSION = 1;

async function initDB(): Promise<IDBDatabase> {
  return new Promise((resolve, reject) => {
    const request = indexedDB.open(DB_NAME, DB_VERSION);

    request.onupgradeneeded = (event) => {
      const db = (event.target as IDBOpenDBRequest).result;

      // 创建 cookies 表
      if (!db.objectStoreNames.contains('cookies')) {
        const store = db.createObjectStore('cookies', { keyPath: 'id' });
        store.createIndex('isActive', 'isActive', { unique: false });
        store.createIndex('healthScore', 'healthScore', { unique: false });
      }
    };

    request.onsuccess = () => resolve(request.result);
    request.onerror = () => reject(request.error);
  });
}
```

## 🧠 核心算法

### 1. 智能选择策略

#### 算法流程

```typescript
getNextCookie(): string | null {
  // Step 1: 过滤活跃 Cookie
  const activeCookies = Array.from(this.cookies.values())
    .filter(c => c.isActive);

  if (activeCookies.length === 0) {
    return null;
  }

  // Step 2: 按健康评分降序排序
  activeCookies.sort((a, b) => b.healthScore - a.healthScore);

  // Step 3: 取前 5 个高质量 Cookie
  const topCookies = activeCookies.slice(0, 5);

  // Step 4: 加权随机选择
  const totalScore = topCookies.reduce((sum, c) => sum + c.healthScore, 0);
  let random = Math.random() * totalScore;
  let selected = topCookies[0];

  for (const cookie of topCookies) {
    random -= cookie.healthScore;
    if (random <= 0) {
      selected = cookie;
      break;
    }
  }

  // Step 5: 更新最后使用时间
  selected.lastUsedAt = Date.now();

  return selected.value;
}
```

#### 算法优势

✅ **负载均衡** - 避免总是使用同一个 Cookie  
✅ **质量优先** - 高分 Cookie 被选中概率更大  
✅ **随机性** - 降低被检测为自动化脚本的风险

### 2. 健康评分算法

#### 计算公式

```typescript
function calculateHealthScore(cookie: CookieEntry): number {
  const now = Date.now();
  const ageInHours = (now - cookie.createdAt) / (1000 * 60 * 60);

  // 基础分 100
  let score = 100;

  // 根据成功率调整
  const totalRequests = cookie.successCount + cookie.failureCount;
  if (totalRequests > 0) {
    const successRate = cookie.successCount / totalRequests;
    score *= successRate;
  }

  // 根据年龄衰减（超过 24 小时每小时减 1 分）
  if (ageInHours > 24) {
    score -= ageInHours - 24;
  }

  // 限制在 0-100 范围
  return Math.max(0, Math.min(100, score));
}
```

#### 影响因素

| 因素     | 权重 | 说明                   |
| -------- | ---- | ---------------------- |
| 成功率   | 高   | 成功次数 / 总请求次数  |
| 年龄     | 中   | 超过 24 小时后开始衰减 |
| 使用频率 | 低   | 间接通过成功率体现     |

### 与 fetch 集成

#### 包装函数

```typescript
// src/utils/fetchWithRetry.ts
export async function fetchWithCookieRetry(
  url: string,
  options: RequestInit = {},
  maxRetries: number = 3
): Promise<Response> {
  const cookiePool = CookiePoolManager.getInstance();

  for (let i = 0; i < maxRetries; i++) {
    const cookie = cookiePool.getNextCookie();

    if (!cookie) {
      throw new Error('没有可用的Cookie');
    }

    try {
      const response = await fetch(url, {
        ...options,
        headers: {
          ...options.headers,
          Cookie: cookie,
        },
      });

      if (response.ok) {
        await cookiePool.reportSuccess(cookie);
        return response;
      }

      await cookiePool.reportFailure(cookie);
    } catch (error) {
      await cookiePool.reportFailure(cookie);

      if (i === maxRetries - 1) {
        throw error;
      }

      // 指数退避等待
      await new Promise((resolve) => setTimeout(resolve, Math.pow(2, i) * 1000));
    }
  }

  throw new Error('所有重试都已失败');
}
```

## 🤖 自动获取功能

### Electron 主进程实现

```typescript
// electron/cookieFetcher.ts
import { BrowserWindow } from 'electron';

export async function fetchCookies(count: number): Promise<string[]> {
  const cookies: string[] = [];

  for (let i = 0; i < count; i++) {
    const cookie = await fetchSingleCookie();
    if (cookie) {
      cookies.push(cookie);
    }

    // 发送进度到渲染进程
    mainWindow.webContents.send('cookie-fetch-progress', {
      current: i + 1,
      total: count,
      cookie: cookie,
    });

    await sleep(2000);
  }

  return cookies;
}

async function fetchSingleCookie(): Promise<string | null> {
  return new Promise((resolve) => {
    const win = new BrowserWindow({
      show: false, // 隐藏窗口
    });

    // 拦截响应头
    win.webContents.session.webRequest.onHeadersReceived(
      { urls: ['*://*.eastmoney.com/*'] },
      (details, callback) => {
        const setCookie = details.responseHeaders?.['set-cookie'];
        if (setCookie && setCookie.length > 0) {
          resolve(setCookie.join('; '));
        } else {
          resolve(null);
        }
        callback({ cancel: false });
      }
    );

    win.loadURL('https://data.eastmoney.com');

    setTimeout(() => {
      win.close();
      resolve(null);
    }, 15000);
  });
}
```

async addCookiesBatch(cookieStrings: string[]): Promise<{
  success: number;
  skipped: number;
  failed: number;
}> {
  const results = { success: 0, skipped: 0, failed: 0 };

  for (const cookieStr of cookieStrings) {
    const result = await this.addCookie(cookieStr);
    if (result.success) results.success++;
    else if (result.skipped) results.skipped++;
    else results.failed++;
  }

  return results;
}

  success: boolean;
  skipped: boolean;
  error?: string;
