## 📊 功能概览

### 核心能力

```
Cookie 池管理系统
├── Cookie 池管理（IndexedDB）
├── 健康管理（评分计算）
├── 自动化获取（批量）
└── 数据管理（导出/导入）
```

### 技术特性

- ✅ **智能选择** - 基于健康评分的加权随机算法
- ✅ **持久化存储** - IndexedDB 本地存储
- ✅ **健康监控** - 实时跟踪使用情况
- ✅ **自动轮换** - 请求失败时自动切换
- ✅ **批量获取** - 支持一次性获取多个
- ✅ **数据导出** - 支持导出/导入备份
