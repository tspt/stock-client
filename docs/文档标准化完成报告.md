## 📋 后续维护规范

### 新增功能时

1. **确定所属模块** → 找到对应路由
2. **更新对应文档**
   - 用户功能 → 更新 `01-使用指南.md`
   - 技术实现 → 更新 `02-技术实现.md`
3. **更新 README**
   - 更新快速索引表格
   - 添加更新记录
4. **检查清单**
   - [ ] 没有创建新文档
   - [ ] 使用相对路径
   - [ ] 添加更新记录

### 修改功能时

1. **判断影响范围**
   - 仅 UI 变化 → 更新 `01-使用指南.md`
   - 技术调整 → 更新 `02-技术实现.md`
   - 两者都有 → 同时更新
2. **同步更新 README**
   - 更新快速索引
   - 添加更新记录
